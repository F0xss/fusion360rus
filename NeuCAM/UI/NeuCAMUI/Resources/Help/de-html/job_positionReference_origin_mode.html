Legt den Befestigungspunkt fest, der zum Verbinden des Bauteils mit dem Befestigungspunkt des Tischs verwendet wird.

Ausgewählter Punkt - Wählen Sie einen Punkt auf der Geometrie aus, der im Ansichtsbereich sichtbar ist.

Punkt auf Modellrahmen - Wählen Sie einen Punkt auf dem virtuellen Rahmen des Modells aus.

Punkt auf Spannmittelbox - Wählen Sie einen Punkt auf dem virtuellen Rahmen des Spannmittels aus.

Punkt auf Rohteilrahmen - Wählen Sie einen Punkt auf dem virtuellen Rahmen des Rohteils aus.

Anmerkung: Wenn Punkt auf Spannmittelbox ausgewählt ist, aber keine Spannmittel auf der Registerkarte Setup definiert sind, wird stattdessen der virtuelle Rahmen des Modells verwendet.
