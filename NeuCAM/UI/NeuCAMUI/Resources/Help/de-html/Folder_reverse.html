Ändert die Reihenfolge der Nachbearbeitung der Instanzen im Muster, sodass die ursprüngliche Operation zuletzt ausgegeben wird.

Die ursprüngliche Operation ist diejenige, die Sie kopieren, um Exemplare in einem Muster zu erstellen. Lassen Sie die Option Umkehren deaktiviert, um die ursprüngliche Operation zuerst auszugeben.
