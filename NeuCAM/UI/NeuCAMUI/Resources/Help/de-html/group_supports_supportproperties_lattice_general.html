Das bestimmte Auflagervolumen wird unterteilt in Einheitenzellen, die anschließend mit Gitter-Topologie gefüllt werden. Mithilfe des so erstellten Gerüsts stützen einzelne Stäbe die entsprechenden Unterseiten-Cluster. Wenn möglich und gewünscht, werden ganze Gruppen von acht Zellen (2 x 2 x 2) in einzelne Zellen reduziert.
