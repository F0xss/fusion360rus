Vermeidet die Bearbeitung von Flachbereichen, die kleiner als der angegebene Wert sind.

Der angegebene Wert wird von der Strategie als Durchmesser eines Kreises verwendet, der zum Auswerten der ebenen Flächen verwendet wird. Wenn der Kreis an keiner Stelle auf eine ebenen Fläche passt, wird die Fläche herausgefiltert.

Es ist hilfreich, flache Bereiche herauszufiltern, die lang und dünn sind, z. B. Fasen, wenn das verwendete Werkzeug nicht zum Bearbeiten von Fasen geeignet ist.

1-mm-Fasen

Filterwert von 0.5 mm
Fasen werden bearbeitet

Filterwert von 1.5 mm
Fasen werden vermieden

Anmerkung: Die Filteroption ist nur verfügbar, wenn Geneigte Flachbereiche bearbeiten auf der Registerkarte Multi-Achsen ausgewählt ist.
