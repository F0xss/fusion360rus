Wählen Sie eine flache oder kreisförmige Fläche oder eine Kante aus, um die Ausrichtung der Rotationsachse festzulegen.

Durch Auswahl einer ebenen Fläche wird die Rotationsachsenausrichtung lotrecht zu dieser Fläche festgelegt.

Durch Auswahl einer kreisförmigen Fläche wird die Rotationsachsenausrichtung parallel zur Achse des Mittelpunkts dieser Fläche festgelegt.

Durch Auswahl einer Kante wird die Rotationsachsenausrichtung parallel zu dieser Kante festgelegt.
