Schließt die Stützstrukturen in die 3MF-Datei ein.

Die Stützstrukturen und das Bauteil werden als separate Körper in der 3MF-Datei gespeichert. Auflager können direkt in Fusion 360 oder einer anderen Software generiert werden. Deaktivieren Sie das Kontrollkästchen, wenn Sie Auflager in einer anderen Software erstellen möchten.
