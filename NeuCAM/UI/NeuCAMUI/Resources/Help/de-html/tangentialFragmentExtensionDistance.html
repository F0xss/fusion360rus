Wird bei offenen Konturen verwendet, um den Anfang und das Ende des berechneten Werkzeugwegs zu verlängern. Dadurch wird eine tangentiale, lineare Verlängerung basierend auf dem Winkel des Start- und Endpunkts erstellt. Diese Verlängerung kann in Kombination mit Geometrie - Tangentialer Verlängerungs-Abstand verwendet werden.

Anmerkung: Sie können die Option Rohteilkonturen verwenden, um eine Verlängerung des Werkzeugwegs über das definierte Rohteil oder eine ausgewählte Begrenzung hinaus zu erzwingen. Diese Option ist gut für unregelmäßige Formen geeignet. Wenn Sie für jedes Ende des Schnitts eine andere Verlängerung benötigen, können Sie dies unter Registerkarte Geometrie - Tangentialer Verlängerungs-Abstand einstellen.
