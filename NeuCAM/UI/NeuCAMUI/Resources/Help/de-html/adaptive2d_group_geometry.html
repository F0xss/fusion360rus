Offene oder geschlossene Begrenzungen können bearbeitet werden, indem Sie Flächen, Kanten oder Skizzen auswählen. Sie können Rohteilmaterial von der Innenseite einer Tasche oder der Außenseite eines Schraublochs entfernen.

Schraubloch

Wenn Sie Flächen auswählen, die über Bohrungen oder Taschen verfügen, wird mit Adaptiv davon ausgegangen, dass diese Öffnungen bereits freigeräumt sind, und eine Bearbeitung wird vermieden. Verwenden Sie die Kantenauswahl für Taschen mit Bohrungen oder Taschen innerhalb der Tasche. Wählen Sie für Schraublöcher die äußere Begrenzung des Schraublochs aus, und aktivieren Sie die unten angezeigte Option Rohteilkonturen. Der Werkzeugweg wird zwischen der ausgewählten Begrenzung und dem äußeren Rohteilbereich berechnet.
