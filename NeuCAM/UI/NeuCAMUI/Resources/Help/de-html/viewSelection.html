Gibt an, wie die Werkzeugausrichtung bestimmt wird.

  • WKS verwenden ist die Vorgabeeinstellung. Dabei wird das Werkstück-Koordinatensystem (WKS) des aktuellen Jobs für die Werkzeugausrichtung verwendet.
  • Punkt und Ebene verwenden legt von jedem ausgewählten Punkt und jeder ausgewählten Ebene als Ursprung bzw. Ausrichtung eine bestimmte Werkzeugausrichtung für diese Operation fest.
  • Koordinatensystem verwenden legt anhand eines beliebigen Koordinatensystem-Elements im Modell eine bestimmte Werkzeugausrichtung für diese Operation fest. Hierbei werden sowohl der Ursprung als auch die Ausrichtung des Koordinatensystem-Elements verwendet. Verwenden Sie diese Option, wenn Ihr Modell keinen geeigneten Punkt und keine geeignete Ebene für die gewünschte Operation enthält.
