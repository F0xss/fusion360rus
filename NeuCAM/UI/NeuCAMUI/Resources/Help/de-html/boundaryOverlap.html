Wenn eine Neigungsbegrenzung und/oder Restmaterialbearbeitung für eine HSC-Kontur-Operation verwendet wird, können Sie das Kontrollkästchen Flächen mit Begrenzungen bearbeiten aktivieren, um von den durch die Neigung und/oder die Restmaterialerkennung erstellten Begrenzungen einen Versatz des Werkzeugwegs zu erstellen.

Sie können eine Begrenzungsüberlappung angeben, sodass die Begrenzung des ermittelten Neigungs- bzw. Restmaterialbereichs um den angegebenen Überlappungswert vergrößert wird.
