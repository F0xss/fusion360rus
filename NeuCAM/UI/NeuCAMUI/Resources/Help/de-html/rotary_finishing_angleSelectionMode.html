Wählen Sie einen Modus aus, um den Endpunkt der schrägen Begrenzung des Werkzeugwegs anzugeben.

Volle 360 - Wählen Sie diese Option aus, um das gesamte Bauteil zu bearbeiten, wobei der erste Durchgang vom ausgewählten Startpunkt aus beginnt.
Endpunkt auswählen - Wählen Sie diese Option aus, damit Sie auf einen Punkt im Modell klicken können, um den Endpunkt für die Winkelbegrenzung anzugeben.
Endwinkel angeben - Wählen Sie diese Option aus, damit Sie einen Winkel eingeben können, der von dem unter Startpunkt angegebenen Punkt aus gemessen wird.
