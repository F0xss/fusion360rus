Gibt an, wie viel Rohmaterial am Modell übrig ist.

Überschüssiges Material und Kollisionen werden von diesem Wert aus über der tatsächlichen Oberfläche des Modells gemessen. Ein Wert von null zeigt Abweichungen direkt von der Modelloberfläche aus an.

Gilt bei Verwendung von Farbdarstellung - Vergleich.
