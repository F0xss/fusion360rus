Wenn diese Option aktiviert ist, werden bei Auswahl einer Fläche aus allen Bauteilen innerhalb der Rohteilbegrenzung alle Flächen auf der gleichen Ebene gesucht.

Wählen Sie eine beliebige Fläche auf einem beliebigen Bauteil für eine Übereinstimmung aus.

Anmerkung: Wird verwendet, um Begrenzungen basierend auf der Flächentiefe zu filtern. Verwenden Sie den Filter Krümmungen, um alle Innen- und Außenbegrenzungen zu erkennen.
