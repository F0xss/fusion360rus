Legt fest, ob das Modell so behandelt wird, als hätte es nur Flachbereiche, nur Steilbereiche oder eine Kombination aus beiden.

Steil und flach - Erkennt, dass das gesamte Modell Steil- und Flachbereiche enthält. Ein Winkelschwellenwert bestimmt, welche Bereiche steil und welche flach sind.

Nur Flachbereich - Erkennt, dass das gesamte Modell Flachbereiche enthält. Sie können damit einen parallelen oder HSC-Kontur-Werkzeugweg erstellen und auf zusätzliche Funktionen zugreifen, wie z. B. die Möglichkeit, lotrechte Durchgänge zu optimieren.

Nur Steilbereiche - Erkennt, dass das gesamte Modell Steilbereiche enthält. Sie können damit einen Kontur-Werkzeugweg erstellen und auf zusätzliche Funktionen zugreifen, wie z. B. die Möglichkeit, Hinterschnitte zu bearbeiten.
