Neigt das Werkzeug in Schnittrichtung mit einem positiven Winkel, der als Ein-/Ausfahrwinkel bezeichnet wird. Sie können das Werkzeug auch von der Schnittrichtung weg neigen, indem Sie einen negativen Winkel verwenden. Dieser wird als Folgewinkel bezeichnet.

Dies ist nützlich, um das Werkzeug zu neigen, um beispielsweise das Schneiden mit dem Mittelpunkt eines Kugelfräsers zu vermeiden.
