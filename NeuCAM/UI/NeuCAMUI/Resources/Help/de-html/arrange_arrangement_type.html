Legt fest, wie die ausgewählten Komponenten im Grundvolumen der additiven Maschine angeordnet werden.

Wenn das Setup eine additive Maschine mit FFF-, SLA- oder MPBF-Funktionen verwendet, können Sie die Anordnung nur in 2D, d. h. auf der Grundplatte, vornehmen. Wenn das Setup eine additive Maschine mit MJF-, SLS- oder Freistrahl-Bindemittelauftrags-Funktionen verwendet, können Sie die Anordnung in 2D oder 3D vornehmen.
