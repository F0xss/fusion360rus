Gibt Zonen auf dem Bauteil an, in denen Haltestege nicht automatisch platziert werden können.

Klicken Sie auf zwei Punkte auf dem Profil des Bauteils, um eine Zone ohne Haltestege festzulegen. Der kürzeste Abstand zwischen den beiden Punkten wird zur Zone ohne Haltestege. Falls erforderlich, können Zonen ohne Haltestege einander überlappen.

Wenn Haltestege an einem Bauteil vorhanden sind, kann dies bedeuten, dass weitere Arbeiten wie Schleifen, Schmirgeln oder Feilen erforderlich sind, um das Bauteil zu bereinigen. Sie können z. B. die Zonen ohne Haltestege verwenden, um Bereiche anzugeben, in denen Sie keine Haltestege platzieren möchten, da aufgrund bestimmter Geometrie die anschließend erforderliche Bereinigung schwierig sein kann.

Tipp: Wenn ein Haltesteg automatisch in einem nicht gewünschten Bereich platziert wird, können Sie diesen Bereich als Zone ohne Haltestege definieren, sodass dort keine Haltestege platziert werden. Stattdessen können Sie in diesem Bereich manuell Haltestege platzieren und dann die Position des Haltestegs manuell anpassen, sodass eine Platzierung im gewünschten Bereich erfolgt.
