Gibt die Richtung der Fräsdurchgänge an.

Durchgangsrichtung 0 Grad

Durchgangsrichtung 45 Grad

Ein häufiger Grund für die mögliche Änderung der Durchgangsrichtung ist die Erzielung einer höheren Effizienz. So kann beispielsweise eine dreieckige Fläche mit weniger Durchgängen bearbeitet werden, wenn die Durchgänge entlang der Richtung der längsten Kante verlaufen, anstatt in eine beliebige Richtung.
