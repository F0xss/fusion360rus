Bevorzugter Winkel für 4-Achsen-Bearbeitung:

Neigt das Werkzeug in einem Winkel zur Rotationsachse hin, der von der Z-Achse der Werkzeugausrichtung aus gemessen wird.

Bevorzugter Winkel für 5-Achsen-Bearbeitung:

Neigt das Werkzeug um den angegebenen Winkel zur Z-Achse der Werkzeugausrichtung. Ein Bevorzugter Neigungswinkel von 0 richtet das Werkzeug an der Z-Achse der Werkzeugausrichtung aus.

Diesen Winkel kann man sich als konische Begrenzung um die Z-Achse der Werkzeugausrichtung vorstellen.
