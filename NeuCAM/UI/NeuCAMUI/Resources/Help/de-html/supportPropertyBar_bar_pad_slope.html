Gibt den Winkel der Verjüngung von der Unterlage zum Stabauflager an.

Der Winkel wird von der Horizontalen aus gemessen. Geben Sie einen Wert zwischen -60 und 60 Grad ein. Je nach dem im Setup ausgewählten Typ der additiven Maschine möchten Sie möglicherweise eine negative oder positive Verjüngung erhalten. Für SLM (selektives Laserschmelzen) kann eine negative Verjüngung bedeuten, dass die Auflager leichter von der Grundplatte entfernt werden können, wenn ein Werkzeug wie z. B. ein Schaber verwendet wird.

Negative Verjüngung (links), keine Verjüngung (Mitte) und positive Verjüngung (rechts).
