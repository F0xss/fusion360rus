Wählen Sie die Fläche aus, um den Werkzeugweg auf die ausgewählte Fläche anzuwenden.

Sie können mehrere Flächen auswählen, um einen einzelnen Fließwerkzeugweg zu generieren. Die Anzahl der angegebenen Querzustellungen wird auf jede Flächenauswahl angewendet.

Anmerkung: Durch Klicken auf den bidirektionalen roten Pfeil wechseln Sie die isometrische Richtung des Werkzeugwegs. Diese kann auch über die Registerkarte Durchgänge dieses Dialogfelds gesteuert werden.
