Konturdurchgänge werden normalerweise von oben nach unten durchgeführt. Aktivieren Sie dieses Kontrollkästchen, um anzugeben, dass Durchgänge von unten nach oben durchgeführt werden.

Die Reihenfolge bewirkt, dass zuerst die Durchgänge mit der kleinsten Werkzeugausrichtung auf der Z-Ebene in einer Operation für mehrere Konturen bearbeitet werden. Diese Methode ist besonders nützlich, um brüchige Materialien wie Graphit zu bearbeiten.
