Unterdrückt die Bearbeitung einer oder mehrerer Nuten.

Nutunterdrückung ist deaktiviert

Nutunterdrückung ist aktiviert

Mit dieser Option können Sie eine oder mehrere Nuten festlegen, die bei der Bearbeitung unterdrückt werden sollen. Legen Sie die Nut fest, indem Sie die Flächen auswählen, die die Nut einschließen. Um diese Option zu verwenden, aktivieren Sie die Option Rotationsgeometrie im Dialogfeld Setup auf der Registerkarte Setup.
