Hilft bei der Bestimmung der Positionen, an denen das Werkzeug einfahren soll, d. h. die Bewegung vor Beginn einer Schnittbewegung.

Wählen Sie Punkte auf dem Modell aus, um die Einfahrpositionen zu bestimmen. Sie können eine oder mehrere Positionen auswählen. Wenn Sie mehrere Positionen auswählen, wird die Position, die am nächsten zu einer Kontur liegt, als ungefähre Einfahrposition verwendet.

Position 1 - Verwendet von einer Kontur
Position 2 - Verwendet von mehreren Konturen

Anmerkung: Die von Ihnen angegebenen Positionen sind Näherungswerte und entsprechen möglicherweise nicht immer der Position, an der das Werkzeug einfährt.
