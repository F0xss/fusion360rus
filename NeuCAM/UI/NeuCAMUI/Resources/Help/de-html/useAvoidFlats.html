Erkennt automatisch Flachbereiche des Modells und vermeidet die Erstellung von Werkzeugwegen über diese. Flachbereiche werden erkannt, wenn sie senkrecht zur Z-Achse der Werkzeugausrichtung liegen.

Wenn Sie ein Werkzeug verwenden, das nicht für Flachbereiche geeignet ist, wie z. B. einen Kugelkopffräser, möchten Sie die Bearbeitung von Flachbereichen möglicherweise vermeiden. Die Flachbereiche können mit einer nachfolgenden Operation mit einem eher geeigneten Werkzeug, wie z. B. einem Schaftfräser, bearbeitet werden.
