Hier können Sie die Reihenfolge auswählen, in der die angegebenen Kontaktpunkte angetastet werden.

Sie können die Punkte in der Reihenfolge antasten, in der Sie sie erstellt haben. Oder Sie können eine optimierte Reihenfolge auswählen, die die Entfernung minimiert, die der Taster entlang der X- und Y-Achse zurücklegt. Wenn Sie eine optimierte Reihenfolge auswählen und einen Punkt erstellen, neu positionieren oder löschen, wird die Reihenfolge automatisch aktualisiert.
