Gibt die Referenzbearbeitungsoperation an, die zum Aktualisieren der Werkzeugverschleißkompensation verwendet wird.

Wählen Sie im Browser die Bearbeitungsoperation aus, für die Sie das Werkzeug aktualisieren möchten. Dies sollte die Operation sein, mit der gerade das Element bearbeitet wurde, das Sie antasten. Wenn für diese Antastoperation ein anderer Antasttyp als Z-Oberfläche verwendet wird, muss für die Referenzbearbeitungsoperation der Kompensationstyp Steuern, Verschleiß oder Umgekehrter Verschleiß verwendet werden.
