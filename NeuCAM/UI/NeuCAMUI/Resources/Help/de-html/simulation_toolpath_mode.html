Steuert, welche Abschnitte der Schnittbewegungen in welcher Reihenfolge angezeigt werden.

Alle Werkzeugwege - Zeigt alle ausgewählten Werkzeugwege an.
Vor Positionierung - Die Werkzeugwegpositionen vor der Bewegung werden angezeigt.
Nach Positionierung - Der Werkzeugweg wird vor der Bewegung angezeigt. Durch die Bewegung wird der Weg gelöscht.
Operation - Zeigt nur die aktuelle Operation an.
Verlängerung - Zeigt eine Verlängerung von der letzten Position an.
