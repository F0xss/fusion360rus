Ein Prozentsatz der Querzustellung zwischen 0 % und 25 %, um den der Werkzeugweg von der festgelegten Querzustellung abweichen kann, um scharfe Ecken im Werkzeugweg zu reduzieren.

Die Glättung gilt für Innenschnitte und wirkt sich nicht auf den endgültigen Profilschnitt aus.

Glättungsabweichung = 0 %

Glättungsabweichung = 25 %
