Sorgt dafür, dass die Werkzeugachsenneigung mit dem bevorzugten Winkel beibehalten wird, wenn dies möglich ist.

Wenn das Werkzeug bei Bearbeitung einer steilen Wand ständig unnötigerweise den Neigungswinkel ändert, können unerwünschte Markierungen auf dem Bauteil entstehen. Aktivieren Sie Neigungswinkel überschreiben, und geben Sie einen bevorzugten Winkel an, damit ein konstanter Neigungswinkel beibehalten und so eine bessere Oberflächengüte erzielt wird und die Risiken für eine Kollision verringert werden.

Keine Überschreibung
Werkzeugneigung frei

Neigungswinkel überschreiben verwendet
Werkzeug verwendet einen bevorzugten Winkel

Anmerkung: Das Werkzeug verwendet den bevorzugten Winkel, und der Neigungswinkel wird nur geändert, wenn eine Kollision vermieden werden muss (wenn Kollisionsvermeidung aktiviert ist).
