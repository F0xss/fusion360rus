Gibt die Kantenlänge für die Voxel oder Elemente an, die bei der Berechnung der Anordnung zur Darstellung der Komponenten verwendet werden.

Je kleiner die Voxelgröße, desto genauer wird die Form der einzelnen Komponenten dargestellt, sodass der verfügbare Platz besser genutzt werden kann.

Anmerkung: Die Berechnung der Anordnung kann länger dauern, wenn eine kleinere Voxelgröße verwendet wird.
