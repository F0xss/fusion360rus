Öffnet die Werkzeugbibliothek, in der Sie eine Düse für Ihren Werkstücktyp auswählen können.

Wählen Sie das passende Werkzeug/die passende Düse für Ihren Schneidprozess aus.

Passende Werkzeug-/Düsentypen
Wasserstrahl-Schneiden
Laserschneiden
Plasmaschneiden

Sie können auch das Symbol für
Neuer Wasserstrahl/Plasma/Laser
auswählen, um ein benutzerdefiniertes Werkzeug/eine benutzerdefinierte Düse zu erstellen.

Anmerkung: Einige der Werkzeugwegoptionen hängen vom Werkzeug-/Düsentyp ab, den Sie auswählen.
