Wendet für die Prozesssimulation in der 3MF-Datei auf verschiedene Stützstrukturen verschiedene Volumenanteile an.

Die 3MF-Datei enthält zwei Auflagernetze für jeden Körper. Das erste ist das echte Auflagernetz, wie es in der Software zu sehen ist, und das zweite ist ein Ersatz-Auflagernetz nur für die Prozesssimulation. Deaktivieren Sie das Kontrollkästchen, um einen einheitlichen Volumenanteil auf alle Ersatznetze anzuwenden. Dies ist beispielsweise der Fall, wenn Sie den gleichen Stil von Stützstrukturen oder den gleichen Stil mit leicht unterschiedlichen Einstellungen verwenden.
