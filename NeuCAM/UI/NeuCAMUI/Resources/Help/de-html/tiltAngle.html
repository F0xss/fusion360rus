Behält die Werkzeugachse im festgelegten bevorzugten Winkel bei.

Der bevorzugte Winkel wird von der angegebenen Werkzeugausrichtung aus gemessen (wenn Sie keine Werkzeugausrichtung auf der Registerkarte Geometrie angeben, wird der bevorzugte Winkel von der Setup-Z-Achse aus gemessen). Wenn Sie die Werkzeugachse in einem konstanten Winkel beibehalten, erhalten Sie eine bessere Oberflächengüte, und der programmierte Vorschub kann eingehalten werden.

Den bevorzugten Winkel kann man sich als konische Begrenzung um die angegebene Werkzeugausrichtung vorstellen.

Bevorzugter Winkel, gemessen von der Z-Achse der Werkzeugausrichtung

Um zu bestimmen, wo sich das Werkzeug auf der konischen Begrenzung befindet, wird dem Werkzeug mit den nicht-vertikalen Primärmodus-Einstellungen eine anfängliche Neigungsrichtung vorgegeben. Dies entspricht dem Verhalten, wenn die Option Neigungswinkel überschreiben nicht verwendet wird. In der anfänglichen Neigungsrichtung wird die Werkzeugachse so verschoben, dass sie an der Kante der Begrenzung ausgerichtet ist, der sie am nächsten liegt.
