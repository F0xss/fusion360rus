Legt die Drehzahl der Spindel fest, die normalerweise in Umdrehungen pro Minute (U/min) angegeben wird. Die Spindeldrehzahl wird im NC-Code als S-Code ausgedrückt.

Das Bohren mit einer Spindeldrehzahl, die mithilfe der Empfehlungen des Werkzeugherstellers festgelegt wird, hilft Ihnen dabei, eine gute Oberflächengüte für Ihre Bohrungen sicherzustellen. Sie können manuell eine Spindeldrehzahl eingeben oder die Spindeldrehzahl von Fusion 360 berechnen lassen, wenn Sie einen Wert für die Schnittgeschwindigkeit eingeben.

Anmerkung: Sie sollten die Spindeldrehzahl möglicherweise reduzieren, je nachdem, ob Sie eine Maschine mit geringer Leistung, wenig Kühlmittel oder Spannvorrichtungen mit niedrigem Klemmdruck verwenden.
