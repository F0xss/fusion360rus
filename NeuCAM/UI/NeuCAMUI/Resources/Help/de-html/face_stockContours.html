Für den Planen-Werkzeugweg ist keine Auswahl erforderlich. Das System plant das Rohteil, das in den Job-Setup-Parametern definiert wurde, automatisch.

Sie können zum Planen einen Bereich mit beliebiger Größe oder Form auswählen. Dieser Bereich kann eine Kanten- oder Skizzenauswahl sein. Der Rohteilbereich für die Bearbeitung wird gelb dargestellt.
