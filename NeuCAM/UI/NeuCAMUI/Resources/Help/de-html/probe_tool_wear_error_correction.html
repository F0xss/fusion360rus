Gibt als Prozentsatz der Werkzeugabweichung an, wie weit die Werkzeugverschleißkompensation aktualisiert wird.

Geben Sie einen Wert zwischen 0 und 100 ein. Beispiel: Wenn die Abweichung eines Werkzeugs 0.05 mm beträgt und die Fehlerkorrektur 80 % lautet, wird die Werkzeugverschleißkompensation auf 0.04 mm aktualisiert.
