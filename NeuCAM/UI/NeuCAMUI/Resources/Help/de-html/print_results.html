Ermöglicht die Ausgabe der Antastergebnisse auf der Maschinenanzeige, in einer Datei auf der Maschinensteuerung oder auf einem externen Gerät.

Diese Funktion hängt von den Funktionen des Postprozessors, der Maschinensteuerung und den Makros des Antastzyklus der Maschine ab. Häufig werden die Daten in einer Datei auf dem lokalen System gespeichert.
