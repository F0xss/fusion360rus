Steuert das Ausmaß, in dem ein Werkzeugweg basierend auf den Begrenzungsbedingungen gestutzt wird.

Werkzeugzentrum auf Begrenzung - Stutzt den Werkzeugweg so, dass sich das Werkzeugzentrum auf der Begrenzung befinden kann. Diese Einstellung stellt sicher, dass die gesamte Fläche innerhalb der Begrenzung bearbeitet wird. Bereiche außerhalb der Begrenzung können jedoch auch bearbeitet werden.

Werkzeug innerhalb der Begrenzung - Stutzt den Werkzeugweg so, dass das gesamte Werkzeug innerhalb der Begrenzung bleibt. Daher wird möglicherweise nicht die gesamte innerhalb der Begrenzung liegende Fläche bearbeitet.

Werkzeug außerhalb der Begrenzung - Stutzt den Werkzeugweg so, dass sich die Werkzeugkante auf der Außenkante der Begrenzung bewegen kann.
