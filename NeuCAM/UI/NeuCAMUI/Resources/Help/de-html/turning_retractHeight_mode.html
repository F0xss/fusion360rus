Mit dem Modus Rückzugshöhe wird die Höhe festgelegt, auf die sich das Werkzeug bewegt, bevor der nächste Schnittdurchgang ausgeführt wird. Die Rückzugshöhe sollte einen Wert über dem Wert für Vorschubhöhe und Oberkante aufweisen. Sie wird zusammen mit dem nachfolgenden Versatz zum Festlegen der Höhe verwendet.

Rückzugshöhe
  • Sicherheitshöhe: Inkrementeller Versatz von der Sicherheitshöhe
  • Obere Höhe: Inkrementeller Versatz von der oberen Höhe.
  • Endhöhe: Inkrementeller Versatz von der Endhöhe.
  • Modelloberkante: Inkrementeller Versatz von der Modelloberkante.
  • Modellunterkante: Inkrementeller Versatz von der Modelloberkante.
  • Rohteiloberkante: Inkrementeller Versatz von der Rohteiloberkante.
  • Rohteilunterkante: Inkrementeller Versatz von der Rohteilunterkante.
  • Ausgewählte Kontur(en): Inkrementeller Versatz von einer im Modell ausgewählten Kontur.
  • Auswahl: Inkrementeller Versatz von einem Punkt (Scheitelpunkt), einer Kante oder Fläche, der bzw. die im Modell ausgewählt wurde.
  • Ursprung (absolut): Absoluter Versatz vom Ursprung, der im Dialogfeld Setup oder in der Gruppe Werkzeugausrichtung dieses Dialogfelds definiert wurde.
