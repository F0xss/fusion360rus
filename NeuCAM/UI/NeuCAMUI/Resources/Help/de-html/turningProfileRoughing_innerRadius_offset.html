Verwenden Sie diesen Versatz, um die Position relativ zum oben ausgewählten Referenzpunkt zu ändern. Sie können nach Bedarf positive oder negative Anpassungen vornehmen. Tipp: Klicken Sie in die farbige Leiste, und ziehen Sie den Mauszeiger.

In diesem Beispiel:
Außen = Rohteil-AD
Innen = Modell-ID

In diesem Beispiel:
Außen = Rohteil-AD
Innen = Modell-ID + 0.50 Zoll Versatz.
Abstand = Innen -0.25 Zoll Versatz.
