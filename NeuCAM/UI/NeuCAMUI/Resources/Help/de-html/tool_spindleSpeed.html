Legt die Drehzahl der Spindel fest, die normalerweise in Umdrehungen pro Minute (U/min) angegeben wird. Die Spindeldrehzahl wird im NC-Code normalerweise als S-Code ausgedrückt.

Das Schneiden mit einer Spindeldrehzahl, die mithilfe der Empfehlungen des Werkzeugherstellers festgelegt wird, sorgt für eine gute Oberflächengüte bei Bauteilen. Sie können eine Spindeldrehzahl manuell eingeben oder die Spindeldrehzahl von Fusion 360 berechnen lassen, wenn Sie eine empfohlene Schnittgeschwindigkeit eingeben.

Anmerkung: Sie sollten die Spindeldrehzahl möglicherweise reduzieren, je nachdem, ob Sie eine Maschine mit geringer Leistung, wenig Kühlmittel oder Spannvorrichtungen mit niedrigem Klemmdruck verwenden.
