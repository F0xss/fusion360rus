Gibt an, wie unbearbeitete Bereiche eines Bauteils identifiziert werden sollen, in denen noch Rohteilmaterial zu entfernen ist. Der resultierende Werkzeugweg zielt nur auf unbearbeitete Bereiche ab.

  • Von vorherigen Operationen - Verwendet alle Werkzeugwege im aktuellen Setup, um die verbleibenden unbearbeiteten Bereiche zu identifizieren.
  • Von Werkzeug - Ermöglicht das Definieren eines Werkzeugs und identifiziert anschließend Bereiche, die mit dem definierten Werkzeug nicht erreichbar sind.
  • Von Datei: Ermöglicht die Angabe einer binären STL-Datei, die unbearbeitete Bereiche darstellt.
  • Von Volumenkörper(n) - Ermöglicht das Angeben von Volumenkörpern, die unbearbeitete Bereiche darstellen.
  • Von Setup-Rohteil - Verwendet das im Setup definierte Rohteil, um unbearbeitete Bereiche darzustellen.
