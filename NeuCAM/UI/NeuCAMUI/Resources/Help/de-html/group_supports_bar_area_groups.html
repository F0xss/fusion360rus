Steuert, ob Stabauflager gruppiert werden, sodass sie verjüngt werden können, einschließlich Grundplattenauflager und Streben. Wenn diese Option aktiviert ist, werden Stabauflager gruppiert. Wenn sie deaktiviert ist, werden Stabauflager nicht gruppiert.

Grundplattenauflager können die Haftung an der Grundplatte verbessern und deren Verschleiß reduzieren. Streben können dabei helfen, eine Knickung der Stabauflager während des Bauprozesses zu verhindern.
