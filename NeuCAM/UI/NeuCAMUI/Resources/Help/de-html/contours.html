Wählen Sie eine Fläche, eine Kante oder eine Skizze zum Definieren der Bearbeitungsbegrenzung aus. Bei Auswahl einer Fläche werden Werkzeugwege auf allen Kanten erstellt. Verwenden Sie die Kantenauswahl für Bereiche mit Bohrungen oder Taschen auf der Fläche. Wenn Sie die untere Kante auswählen, wird die Referenz für die Schnitttiefe automatisch festgelegt.
