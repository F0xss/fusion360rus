Bestimmt, ob Taschen manuell mithilfe von Ausgewählte Taschen oder automatisch anhand ihrer Geometrie mithilfe von Taschenerkennung ausgewählt werden.

Ausgewählte Taschen - Verwenden Sie diese Option, wenn Sie ein Bauteil mit geringer Komplexität bearbeiten, das nur über wenige Taschen verfügt.

Taschenerkennung - Verwenden Sie diese Option, wenn Sie ein komplexes Bauteil mit mehreren Taschen unterschiedlicher Größe bearbeiten. Die Taschenerkennung kann Ihnen Zeit sparen, indem Taschen automatisch auf Basis ihrer Geometrie ausgewählt werden. Geben Sie die Geometriebegrenzungen basierend auf der Geometrie des Werkzeugs an.

Anmerkung: Bei der Taschenerkennung werden nur geschlossene Taschen berücksichtigt. Die geschlossenen Taschen werden an ihren Kanten und nicht an ihren Flächen ausgewählt.
