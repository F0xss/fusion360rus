Aktivieren Sie diese Option, um das Schruppen als festen Zyklus für Ihre Steuerung auszugeben.

Dies kann die Menge an Code verringern, der für das NC-Programm generiert wird. Außerdem werden einige der erweiterten Optionen für den Profil-Schruppen-Werkzeugweg beschränkt, da die meisten festen Zyklen nur einen einfachen Begrenzungsschnitt erstellen.

Anmerkung: Dazu müssen möglicherweise Änderungen des Postprozessors speziell für Ihre Maschinensteuerung vorgenommen werden.
