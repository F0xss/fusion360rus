Öffnet die Druckeinstellungsbibliothek, in der Sie die Druckeinstellung auswählen können, die mit der ausgewählten additiven Maschine verwendet werden soll.

Eine Druckeinstellung ist eine Gruppierung materialspezifischer Parameter, wie z. B. die Layerhöhe, die Sie mit einer additiven Maschine verknüpfen. Sie definiert, wie ein Bauteil additiv gefertigt wird. In der Bibliothek mit Druckeinstellungen können Sie Druckeinstellungen für additive Maschinen erstellen, bearbeiten, kopieren und einfügen sowie importieren und exportieren. Eine Druckeinstellung muss ausgewählt werden, bevor Sie das additive Setup erstellen können.
