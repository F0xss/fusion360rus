Diese Option wird in Hellblau angezeigt und legt die Referenzposition für den Außenradius fest. Die Referenz kann in Bezug zum Rohteil, zum Modell, einem angegebenen Radius, Durchmesser oder einer der anderen radialen Positionen stehen. Diese Referenzposition kann mit einem positiven oder negativen Versatzwert verschoben werden.

Abstand - Legt den Außenradius in Bezug zur Abstandsposition fest.

Rückzug - Legt den Außenradius in Bezug zur Rückzugsposition fest.

Rohteil-OD - Legt den Außenradius in Bezug auf den Außendurchmesser des definierten Rohteils fest.

Modell-OD - Legt den Außenradius in Bezug auf den Außendurchmesser des definierten Modells fest.

Außenradius - Legt den Außenradius in Bezug auf die Position des Außenradius fest. Der Abstandsradius muss größer sein als der Außenradius. Verwenden Sie den Parameter Versatz, um bei Bedarf Anpassungen vorzunehmen.

Innenradius - Legt den Außenradius in Bezug auf die Position des Innenradius fest. Der Abstandsradius muss größer sein als der Innenradius. Verwenden Sie den Parameter Versatz, um bei Bedarf Anpassungen vorzunehmen.

Modell-ID - Legt den Außenradius in Bezug auf den Innendurchmesser des Modells gemäß Definition im Setup fest. Verwenden Sie den Parameter Versatz, um bei Bedarf Anpassungen vorzunehmen.

Rohteil-ID - Legt den Außenradius in Bezug auf den Innendurchmesser des Rohteils gemäß Definition im Setup fest. Verwenden Sie den Parameter Versatz, um bei Bedarf Anpassungen vorzunehmen.

Auswahl - Wählen Sie eine Fläche, einen Scheitelpunkt oder einen Punkt im Modell zur Definition des Abstandsradius aus. Verwenden Sie den Parameter Versatz, um nach Bedarf Anpassungen mit positiven oder negativen Werten vorzunehmen.

Radius - Mit dieser Option können Sie einen Wert für den Radius in das Feld Versatz eingeben. Dieser Wert wird in Bezug auf die Mittellinie des Bauteils festgelegt, und es werden keine assoziativen Änderungen am Modell erkannt.

Durchmesser - Mit dieser Option können Sie einen Wert für den Durchmesser in das Feld Versatz eingeben. Dieser Wert wird in Bezug auf die Mittellinie des Bauteils festgelegt, und es werden keine assoziativen Änderungen am Modell erkannt.

Anmerkung: Der Außenradius muss kleiner als oder gleich dem Rückzugs-, aber größer als der Innenradius sein, um einen gültigen Werkzeugweg zu generieren.
