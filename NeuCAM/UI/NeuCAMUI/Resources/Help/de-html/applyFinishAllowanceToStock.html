Wenn diese Option aktiviert ist, wird das Schlichtaufmaß (definiert durch das Rohteil-Aufmaß) zusätzlich zum Modell auf das Rohteil angewendet. Dies bedeutet, dass der Werkzeugweg nach der Schulter eines Modells nicht abfällt. Stattdessen verfolgt der Werkzeugweg weiter eine gerade Linie und hinterlässt auf dem Rohteil ebenfalls ein Schlichtaufmaß.

Wenn diese Option deaktiviert ist, wird das Schlichtaufmaß nur auf das Modell angewendet.
