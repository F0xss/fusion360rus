Gibt an, um wie viel Grad das Werkzeug in Bewegungsrichtung geneigt werden soll.

Ein positiver Wert neigt das Werkzeug in Richtung der Verfahrrichtung, was häufig als Ein-/Ausfahrwinkel bezeichnet wird. Ein negativer Wert neigt das Werkzeug von der Verfahrrichtung weg, was häufig als Folgewinkel bezeichnet wird.
