Verwendet eine konische anstatt einer kreisförmigen Bearbeitungsebene.

Deaktiviert
Kreisförmige Ebene

Aktiviert
Konische Ebene

Sie können daher konische Flächen bearbeiten, indem Sie das Werkzeug so neigen, dass es lotrecht zur konischen Fläche ist.

Dadurch wird die Anzahl der beim Schruppen erforderlichen Tiefenzustellungen reduziert, und überschüssiges Rohteilmaterial bleibt beim Schlichten von konischen Flächen nicht zurück.

Deaktiviert
Überschüssiges Rohteilmaterial

Aktiviert
Oberfläche vollständig bearbeitet
