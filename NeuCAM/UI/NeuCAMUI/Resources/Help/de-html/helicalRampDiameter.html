Der maximale Durchmesser für eine spiralförmige Anfahrt in die Kavität.

Bei einem optimalen Wert überlappt das Werkzeug die Mitte, während gleichzeitig weiterhin die maximale spiralförmige Bohrung für die Anfahrt in die Kavität erstellt wird. Das Ziel ist eine gute Span-Ausräumung. Wenn der Wert über dem Durchmesserwert des Werkzeugs liegt, verbleibt möglicherweise ein Zapfen in der Mitte der Spirale.

Wert von 1.8 x Durchm.

Wert von 0.8 x Durchm.

Das System bestimmt die beste Position für die Anfahrspirale, es sei denn, die Option Anfahrpositionen ist ausgewählt. (Anfahrpositionen unten dargestellt.) Es wird, unabhängig vom Durchmesser, immer der gesamte Bereich bearbeitet. Anmerkung: Die Breite des Bereichs sollte Werkzeugdurchm. + Rampendurchm. + Rohteil entsprechen. Ansonsten schlägt die spiralförmige Anfahrt fehl.
