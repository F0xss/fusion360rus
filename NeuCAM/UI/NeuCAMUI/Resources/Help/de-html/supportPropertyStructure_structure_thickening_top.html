Wendet eine andere Dicke auf die oberen Verbindungen einer Stützstruktur als auf die Hauptstruktur an.

Obere Verbindungen bezieht sich auf die Schnittstelle zwischen einer Stützstruktur und einem darüber liegenden Körper. Es gibt verschiedene Verbindungstypen, die auf der Registerkarte Verbindungen festgelegt werden können. Eine größere Dicke ermöglicht eine stärkere Verbindung mit dem Bauteil, erschwert jedoch das Entfernen der Stützstruktur.
