Vorschub, mit dem sich das Werkzeug zurück auf die Rückzugshöhe bewegt, bevor es mit dem nächsten Schnittdurchgang fortfährt.

Der Rückzugsvorschub ist mit keinem anderen Wert auf der Registerkarte Werkzeug verknüpft und kann daher unabhängig festgelegt werden. Die Rückzugshöhe kann auf der Registerkarte Höhen definiert werden.
