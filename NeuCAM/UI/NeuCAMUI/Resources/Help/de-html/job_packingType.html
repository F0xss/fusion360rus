Platziert alle ausgewählten Komponenten so im Grundvolumen der additiven Maschine, dass sie sich nicht überschneiden.

Wenn das Setup eine additive Maschine mit FFF-, SLA- oder MPBF-Funktionen verwendet, können Sie nur in 2D, d. h. auf der Grundplatte, anordnen. Wenn das Setup eine additive Maschine mit MJF- oder SLS-Funktionen verwendet, können Sie die Anordnung in 2D oder 3D vornehmen. Damit eine Komponente in das Grundvolumen passt, muss sie mindestens 1 mm freien Platz in der X-, Y- und Z-Richtung haben.
