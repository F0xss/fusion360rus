Ermöglicht die Auswahl des Bearbeitungsbereichs nach kontinuierlichen Flächen, einzelnen Flächen oder Konturpaaren von Ketten.

Ein-/Ausfahrmodus = Flächen
Auswahlmodus = Flächen
Kontinuierliche Flächen auswählen.

Ein-/Ausfahrmodus = Flächen
Auswahlmodus = Manuell
Einzelne Flächen auswählen.

Ein-/Ausfahrmodus = Konturen
Auswahlmodus = Konturpaare
Wählen Sie die untere und obere Kontur aus,
die den 5-Achs-Wälzen-Werkzeugweg steuert.
Wählen Sie die untere Kette zuerst aus.

Ein-/Ausfahrmodus = Konturen
Auswahlmodus = Manuell
Einzelne Kanten auswählen
die den 5-Achs-Wälzen-Werkzeugweg steuert.
Wählen Sie die untere Kette zuerst aus.
