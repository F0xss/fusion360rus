Definiert die Tastergeschwindigkeit während Messbewegungen, d. h., wenn der Taster das Bauteil berührt. Messbewegungen werden im Ansichtsbereich blau angezeigt.

Anmerkung: Der Messvorschub muss mit dem Kalibrierungsvorschub auf der Maschine übereinstimmen, um die bestmögliche Genauigkeit sicherzustellen.
