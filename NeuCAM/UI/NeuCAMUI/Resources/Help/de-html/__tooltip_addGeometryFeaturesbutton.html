Ermöglicht die Auswahl von Geometrie aus dem Ansichtsbereich auf Grundlage von Auswahlfiltern, um die Auswahl zu beschleunigen. Die verfügbaren Auswahlfilter können für jede Strategie unterschiedlich sein.

Tipp: Sie können die Geometrieauswahl aus anderen Operationen wiederverwenden. Erweitern Sie zum Wiederverwenden der Auswahl bei aktivierter Schaltfläche Auswählen im Browser den Knoten einer Operation, und klicken Sie dann auf die Geometrieauswahl.
