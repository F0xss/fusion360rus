Der kleinste zylindrische Durchmesser, der bearbeitet werden kann.

Dies begrenzt Tiefenschnitte basierend auf der Größe der Kavität. Wenn auf null festgelegt, erfolgt die Bearbeitung bis zu den Grenzen der sicheren Einpassung.

Auf null festlegen
Schneidet bis zur vollen Tiefe

Auf 0.320 Zoll festlegen
Größer als, Durchm. Kavität - Durchm. Werkzeug

Tipp: Der Wert wird wirksam, wenn er größer ist als die Differenz zwischen dem Durchmesser der Kavität minus dem Durchmesser des Werkzeugs.
