Steuert, ob Streben zu den Stabauflagern hinzugefügt werden. Wenn diese Option ausgewählt ist, werden Streben hinzugefügt. Wenn sie deaktiviert ist, werden keine Streben hinzugefügt.

Streben können dazu beitragen, eine Knickung von Stabauflagern während des Bauprozesses zu verhindern.
