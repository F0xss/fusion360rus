Legt die maximale Größe eines erkannten Bereichs fest. Wenn die Größe eines erkannten Bereichs über dem angegebenen Wert liegt, werden keine Auflager generiert.

Der Wert sollte mindestens 0.1 cm² betragen. Diese Einstellung ist hilfreich, wenn Sie eine andere Art von Auflager für große Bereiche erstellen möchten. Beispielsweise können für Bereiche, die kleiner als der definierte maximale Bereich sind, Stabauflager verwendet werden, während für alle größeren Bereiche Volumenauflager verwendet werden können.
