Legt die Anzahl der beim Entgraten zu verwendenden Maschinenachsen fest.

Im Allgemeinen können Sie mit der Multi-Achsen-Bearbeitung mehr Bereiche in einer Operation erreichen. Für die 4- und 5-Achsen-Bearbeitung benötigen Sie eine Werkzeugmaschine, die für die Multi-Achsen-Bearbeitung geeignet ist.

  • 3-Achsen-Bearbeitung - Nützlich für das Entgraten von prismatischen Bauteilen, die leicht mit dem Werkzeug erreichbare Kanten aufweisen.
  • 4-Achsen-Bearbeitung - Nützlich für das Entgraten von zylindrischen Bauteilen.
  • 5-Achsen-Bearbeitung - Nützlich für das Entgraten komplexer Bauteile, bei denen das Werkzeug geneigt werden muss, um mehr Bereiche zu erreichen.
