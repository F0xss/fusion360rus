Wählen Sie diese Option aus, um zusätzlich zur im Setup definierten Geometrie andere Körper, Flächen oder Oberflächen in die Werkzeugwegberechnung einzuschließen. Sie können alternativ die im Setup definierte Modellgeometrie überschreiben und stattdessen andere Körper, Flächen oder Oberflächen bearbeiten.

Eine typische Verwendung dieser Option ist, Abschlussflächen einzuschließen, die verhindern, dass das Fräswerkzeug in Bohrungen einfährt, die später im Setup durch Bohroperationen gefertigt werden.
