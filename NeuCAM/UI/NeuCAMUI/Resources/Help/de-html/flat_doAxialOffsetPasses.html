Generiert Durchgänge an mehreren Tiefen.

Wenn die Tiefe des Rohteils relativ zum Werkzeug sehr groß ist, können Sie das Material an mehreren Tiefen schneiden, um jedes Mal eine bestimmte, kleinere Rohteilmenge abzutragen und so eine Überlastung des Werkzeugs zu vermeiden.
