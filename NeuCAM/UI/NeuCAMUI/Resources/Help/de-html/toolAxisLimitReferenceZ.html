Bestimmt, von wo aus die Winkelgrenzen der Maschine gemessen werden.

  • Setup Z - Legt die Winkelgrenzen von der Z-Achse des Setups fest.
  • Werkzeugausrichtung - Legt die Winkelgrenzen von der Z-Achse der Werkzeugausrichtung fest.

Tipp: In Szenarien, in denen Sie die Werkzeugausrichtung ändern, um einen Bereich zu bearbeiten, können Sie Setup Z verwenden, um die Neigungswinkelgrenzen so einzustellen, dass Sie den Werkzeugweg auf vorhersagbare Bereiche begrenzen können.
