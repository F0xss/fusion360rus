Bestimmt die Ausfahrposition des Werkzeugs für eine sicherere Bearbeitung.

Wählen Sie einen Punkt aus, um die Ausfahrposition zu bestimmen. Der Punkt muss so nah wie möglich an dem Bereich liegen, den Sie bearbeiten, z. B. der Eingang einer offenen Tasche, damit der Punkt während der Werkzeugwegberechnung erkannt wird.

Wenn Sie mehr als einen Punkt auswählen, wird das Werkzeug nach Abschluss der Schnittbewegungen vom nächstgelegenen Punkt aus zurückgezogen.

Im folgenden Beispiel wird eine offene Tasche mit einem Hinterschnitt bearbeitet. Zunächst wird das Werkzeug am Ende der Tasche zurückgezogen, was zu einer Kollision mit der direkt darüber liegenden Geometrie führt. Mithilfe einer Ausfahrposition können Sie erzwingen, dass das Werkzeug unten verbleibt und aus der Tasche ausfährt, sodass es mit ausreichendem Abstand zurückgezogen werden kann.

Anmerkung: Das Werkzeug führt eine Flächenkontakt-Verknüpfungsbewegungen durch, wenn es an eine Ausfahrposition verschoben wird. Maximaler Flächenkontaktabstand muss groß genug sein, um diese Bewegung zu ermöglichen. Für 2D-Tasche und 2D-Kontur müssen Sie das Kontrollkästchen Werkzeug unten halten aktivieren, um einen Wert für Maximaler Flächenkontaktabstand einzugeben.
