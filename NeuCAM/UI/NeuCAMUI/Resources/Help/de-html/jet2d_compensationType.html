Gibt den Kompensationstyp an.

  • Im Computer - Die Kopfposition wird vom Programm auf Grundlage des Schnittdurchmessers und der seitlichen Kompensationsrichtung berechnet. Die nachbearbeitete Ausgabe enthält die tatsächliche kompensierte Position, um das Bauteil auf die richtige Größe zuzuschneiden. Der Computer kompensiert die Schnittbreite.
  • An Steuerung - Die Kopfposition wird als tatsächliche Koordinaten der ausgewählten Kette ausgegeben. Die Kompensation wird nicht berechnet, sondern es werden Kompensationscodes ausgegeben, damit der Bediener das Versatzvolumen an der Steuerung des Bearbeitungswerkzeugs festlegen kann. Die Maschinensteuerung kompensiert die Schnittbreite.
