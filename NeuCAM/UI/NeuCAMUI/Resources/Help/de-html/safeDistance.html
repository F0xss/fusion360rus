Legt den Mindestabstand zwischen dem Werkzeug und den Flächen des Bauteils während Rückzugsbewegungen fest. Innerhalb dieses Abstands wird das Werkzeug mit einer direkten Bewegung zurückgezogen, anstatt mit einer Eilgangbewegung.

Um das Werkzeug mit einer Eilgangbewegung vom Bauteil zurückzuziehen, verringern Sie den Wert für den Sicherheitsabstand. Der Abstand wird gemessen, nachdem das Rohteil-Aufmaß angewendet wurde. Wenn ein negatives Rohteil-Aufmaß verwendet wird, stellen Sie sicher, dass der Sicherheitsabstand groß genug ist, um Kollisionen zu vermeiden.
