Gibt die primäre Neigungsmethode des Werkzeugs bei der Bearbeitung des Bauteils an.

Vertikal - Richtet das Werkzeug an der angegebenen Werkzeugausrichtung aus.

Voreil-/Neigungswinkel - Neigt die Werkzeugachse, indem Voreil- und Neigungswinkel angegeben werden. Bei einem Voreil- und Neigungswinkel von 0 wird die Werkzeugachse so ausgerichtet, dass sie immer lotrecht zu der Fläche verläuft, die sie berührt. Ein Voreilwinkel neigt das Werkzeug in Richtung des Schnitts. Ein Neigungswinkel neigt das Werkzeug seitlich von der Schnittrichtung. Ein Beispiel für die Verwendung von Voreil- und/oder Neigungswinkeln ist die Neigung der Werkzeugachse, um ein Schneiden mit der Mitte eines Kugelfräsers zu vermeiden.

Von Kurve - Die Spitze des Werkzeugs zeigt weg von einer skizzierten Kurve.

Zu Kurve - Die Spitze des Werkzeugs zeigt auf eine skizzierte Kurve.

Startpunkt - Die Spitze des Werkzeugs zeigt von einem skizzierten Punkt weg.

Endpunkt - Die Spitze des Werkzeugs zeigt auf einen skizzierten Punkt.
