Hier können Sie die Reihenfolge festlegen, in der die ausgewählten Bohrungen oder Zapfen angetastet werden.

Sie können die Bohrungen oder Zapfen in der Reihenfolge antasten, in der Sie sie ausgewählt haben. Oder Sie können eine optimierte Reihenfolge auswählen, die die Entfernung minimiert, die der Taster zurücklegt. Sie können auch Bohrungen oder Zapfen von der Mitte und nach außen bzw. in der X- oder Y-Richtung antasten. Wenn Sie eine Bohrung oder einen Zapfen auswählen oder die Auswahl aufheben, wird die Reihenfolge automatisch aktualisiert.
