Gibt die Form der Basisfläche an, auf der die Elemente abgelegt werden sollen.

  • Ebene: Eine ebene Fläche in der XY-Ebene.
  • Zylinder: Eine zylindrische Fläche.
  • Rotationsfläche: Fläche, die durch die Rotation einer Kurve um die X-Achse generiert wird.
  • Beliebig: Eine Freiformfläche.

Anmerkung: Wenn die Basisfläche auf einem Zylinder oder einer Rotationsfläche basiert, muss die X-Achse an der Rotationsachse ausgerichtet sein.
