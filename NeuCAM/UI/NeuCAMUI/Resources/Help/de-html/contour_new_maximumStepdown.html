Maximale Tiefenzustellung

Die maximale Tiefenzustellung wird auf die volle Tiefe abzüglich verbleibender Rohteil- und Schlichtdurchlaufmaterialien angewendet. Die abschließende Tiefenzustellung wird auf das verbleibende Rohteil angewendet.
