Wählen Sie eine Kante oder Skizzenbegrenzung aus. Der Werkzeugweg beginnt außerhalb der ausgewählten Begrenzung. Auf diese Weise können Sie einen Werkzeugweg erstellen, der der Kontur des Bauteils möglichst genau entspricht.

Werkzeugweg wird anhand einer
ausgewählten Begrenzung berechnet,
gelb dargestellt

Im obigen Beispiel wird für eine Kontur eine einzelne Kante ausgewählt, und zusätzliche Schruppdurchgänge werden definiert. Der Werkzeugweg wird mit Rohteilkonturen verlängert, sodass die Kanten des Rohteilbereichs freigeräumt werden. Damit kann außerdem die Anzahl der Schruppdurchgänge in diesem Bereich reduziert werden.
