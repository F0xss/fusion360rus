Wendet eine andere Dicke auf die unteren Verbindungen einer Stützstruktur als auf die Hauptstruktur an.

Untere Verbindungen bezieht sich auf die Schnittstelle zwischen einer Stützstruktur und einem darunter liegenden Körper. Es gibt verschiedene Verbindungstypen, die auf der Registerkarte Verbindungen festgelegt werden können. Eine größere Dicke ermöglicht eine stärkere Verbindung mit dem Bauteil, erschwert jedoch das Entfernen der Stützstruktur.
