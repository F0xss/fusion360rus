Gibt die kleinste Abweichung im Werkzeug an, über der die Werkzeugverschleißkompensation aktualisiert wird.

Geben Sie einen positiven Wert ein. Beispiel: Ein Wert von 0.05 mm bedeutet, dass der Werkzeugverschleiß nur aktualisiert wird, wenn die Abweichung des Werkzeugs 0.05 mm oder mehr beträgt. Bei einem Wert von null wird die Kompensation immer aktualisiert.
