Gibt an, wie vorzugehen ist, wenn das Werkzeug auf innere Ecken trifft, die es aufgrund der Werkzeuggeometrie nicht erreichen kann.

Nicht entgraten - Bewegt das Werkzeug zur nächsten Kante der inneren Ecke, nachdem es keine Bearbeitung der aktuellen Kante mehr durchführen kann. Dadurch verbleiben Grate an einigen inneren Ecken. Verwenden Sie diese Option, um eine schnellere Zykluszeit über das Entgraten von mehr inneren Ecken zu priorisieren.

Nicht erreichbare Ecke

Grate an inneren Ecken

Mit Werkzeugspitze schneiden - Versucht, mehr von der inneren Ecke zu schneiden, indem mehr von der Werkzeugspitze verwendet wird. Dadurch wird so viel Material wie möglich von den inneren Ecken entfernt.

Schneiden mit der Werkzeugspitze

Anmerkung: Um die Funktion An festem Punkt auf Werkzeug schneiden zu verwenden, müssen Sie Nicht entgraten auswählen.
