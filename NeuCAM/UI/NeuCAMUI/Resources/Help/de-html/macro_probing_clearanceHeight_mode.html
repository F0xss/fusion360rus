Legt fest, von wo die Sicherheitshöhe versetzt wird. Die Sicherheitshöhe ist die erste Höhe, auf die der Taster auf dem Weg zum Antaststart bewegt wird. Sie ist auch die Höhe, auf die der Taster nach Abschluss des Antastens zurückkehrt.

Wählen Sie einen Versatz von einer anderen Höhe, von der Ober- oder Unterseite des Modells oder Rohteils, von einem ausgewählten Punkt, einer Kante oder Fläche auf dem Modell, von der Oberseite der niedrigsten angetasteten Fläche oder vom Ursprung aus.
