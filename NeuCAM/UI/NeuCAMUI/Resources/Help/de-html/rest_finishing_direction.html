Gibt an, ob Material mittels Gleichlauffräsen, Gegenlauffräsen bzw. einer Kombination aus beiden entfernt wird.

Gegenlauffräsen
  • Beim Gleichlauffräsen wird das Werkzeug in das Bauteil hineingezogen.
  • Beim Gegenlauffräsen wird das Werkzeug vom Bauteil weggeschoben.
  • Bei Beide Richtungen wird die Schnittrichtung abgewechselt.
