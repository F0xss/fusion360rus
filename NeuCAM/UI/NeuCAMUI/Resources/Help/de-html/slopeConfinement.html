Enthält Werkzeugwege basierend auf einem angegebenen Winkelbereich.

0° - 90°

0° - 45°

45° - 90°

Die Neigungswinkel begrenzung wird durch die Parameter Von Neigungswinkel und Bis Neigungswinkel auf der Registerkarte Geometrie bestimmt. Die Winkel reichen von 0° (horizontal) bis 90° (vertikal).

Es werden nur Bereiche bearbeitet, die größer oder gleich den Werten der Parameter Von Neigungswinkel und Bis Neigungswinkel sind.

Die Neigungswinkelbegrenzung wird von den meisten 3D-Bearbeitungsstrategien unterstützt. Beispielsweise dient die Neigungswinkelbegrenzung dazu, eine ausgewählte Werkzeugwegstrategie auf Winkel zu begrenzen, die am besten geeignet sind. So eignet sich Parallele Endbearbeitung besser für Flachbereiche, während Konturendbearbeitung besser für steile Bereiche geeignet ist.
