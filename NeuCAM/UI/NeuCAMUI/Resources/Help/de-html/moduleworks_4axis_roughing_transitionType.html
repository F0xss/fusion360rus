Gibt den Typ der Verbindungsbewegung zwischen Durchgängen an, die auf derselben radialen Ebene nahe beieinander liegen.

  • Kein Kontakt - Verwendet eine Rückzugsbewegung zum Verbinden der Durchgänge. Der Rückzugsabstand kann von der Rückzugsart abhängen.
  • Gerade Linie - Verwendet eine gerade Linie zum Verbinden der Durchgänge.
  • Gleichmäßig - Verwendet gleichmäßige tangentiale Bewegungen zum Verbinden der Durchgänge.
