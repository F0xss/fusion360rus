Gibt den primären Modus der Werkzeugachse an. Wählen Sie je nach Geometrie des Modells die Option aus, mit der Sie die meisten Bereiche mit dem Werkzeug erreichen können.

Primärer Modus für 4-Achsen-Bearbeitung:

  • Zur Rotationsachse - Die Werkzeugspitze zeigt immer zur Rotationsachse.
  • Zur Rotationsachse mit Neigung - Zuerst zeigt die Spitze des Werkzeugs auf die Rotationsachse. Anschließend neigt sich das Werkzeug zur Rotationsachse.
  • Ein-/Ausfahrwinkel von null mit Neigung - Zunächst richtet sich das Werkzeug am durchschnittlichen Winkel zwischen den beiden Kanten aus. Anschließend neigt sich das Werkzeug zur Rotationsachse.

Primärer Modus für 5-Achsen-Bearbeitung:

  • Ein-/Ausfahrt - Neigt das Werkzeug in die Verfahrrichtung.
  • Zu Z neigen - Neigt das Werkzeug an eine beliebige Position auf einer konischen Begrenzung, die um die Z-Achse der Werkzeugausrichtung liegt.
