Wählen Sie eine geschlossene Begrenzung aus, um den Bearbeitungsbereich zu definieren. Sie können eine beliebige Kante aus dem Modell oder eine Skizzenbegrenzung auswählen. Der Bearbeitungsbereich des Rohteils kann dadurch beschränkt oder ausgeweitet werden. Für die Bearbeitung der Rohteilbox, die im Setup angegeben wurde, ist keine Auswahl erforderlich. Wenn Sie eine Begrenzung auswählen, die größer als das Rohteil ist, wird der Schnittbereich verlängert. Dies ist hilfreich bei unregelmäßigen Rohteilgrößen. Die ausgewählte Bearbeitungsbegrenzung kann eine beliebige Form aufweisen.
