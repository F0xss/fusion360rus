Gibt die Reihenfolge der einzelnen Operationen in diesem Setup im NC-Code an.

Reihenfolge beibehalten - Ordnet die Operationen entsprechend dem Layout der Werkzeugwege an, die Sie erstellt haben, und verarbeitet die Operationen nach Setup.

Nach Operation sortieren - Wechselt zwischen mehreren Versätzen, wobei jede Operation für jedes Setup durchgeführt wird, bevor mit der nächsten Operation fortgefahren wird.

Nach Werkzeug sortieren - Ordnet die Operationen an, indem ein Werkzeug geladen wird und alle Operationen für dieses Werkzeug in jedem Setup abgeschlossen werden, bevor mit dem nächsten Werkzeug fortgefahren wird.
