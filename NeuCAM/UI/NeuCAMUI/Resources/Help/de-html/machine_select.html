Zeigen Sie die verfügbaren Maschinentypen an, und wählen Sie eine Maschine aus der Liste aus.

Durch Auswahl des Maschinentyps wird definiert, welche Werkzeugweg-Strategien verfügbar sind. Je nach Auswahl stehen möglicherweise weitere maschinenabhängige Parameter zur Verfügung. Wählen Sie Ihre Maschine aus der Liste aus, oder entscheiden Sie sich für eine exemplarische Maschinendarstellung.

Fräsen - Bearbeitung mit Drehspindeln und 2-5 Bewegungsachsen.

Drehen - 2-Achsen-Drehmaschinen und Drehmaschinen mit Fräs-/Dreh-Konfigurationen.

Schneiden - Wasserstrahl-, Laser- oder Plasmaschneidemaschinen.

Additiv - 3D-Druck mit Kunststoff oder Metall
