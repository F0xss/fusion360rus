Ermöglicht die Angabe, an welchen Rahmentypen das Polylinien-Auflager erstellt werden soll.

Bei der Einstellung Freier Rand wird die Polylinie an Rändern erzeugt, die nicht mit einer Wand zusammentreffen. Bei der Einstellung Rand zu Wand wird die Polylinie an Rändern erzeugt, die mit einer Wand zusammentreffen. Bei der Einstellung Alle wird die Polylinie an allen Rändern erzeugt.
