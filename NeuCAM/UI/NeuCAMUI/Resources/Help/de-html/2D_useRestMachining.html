Beschränkt die Operation auf die Abtragung von Material, das von einem zuvor verwendeten Werkzeug bzw. einer Operation nicht entfernt werden konnte.

Rest steht für REmaining STock (verbleibendes Rohteil).

Zu bearbeitender Bereich.
Tasche grün angezeigt.

Vorherige Operation.
Es wird nicht das gesamte Rohteil entfernt.

Mit einem kleineren Werkzeug können Sie die gesamte Tasche bearbeiten, oder Sie verwenden die Restmaterialbearbeitung, um das verbleibende Rohteil abzutragen.

Restmaterialbearbeitung verwenden AUS.
Alle Bereiche werden bearbeitet.

Restmaterialbearbeitung verwenden EIN.
Zuvor nicht bearbeitete Bereiche werden bearbeitet.

Erfordert zusätzliche Informationen über das Werkzeug, das zuvor zum Schneiden der Begrenzung verwendet wurde.
