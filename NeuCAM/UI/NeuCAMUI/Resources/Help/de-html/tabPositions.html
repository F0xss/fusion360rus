Zusätzlich zu den automatisch generierten Haltestegen können Sie manuelle Haltestege hinzufügen, um das Bauteil während des Bearbeitungsvorgangs in seiner Position zu fixieren.

Klicken Sie auf eine beliebige Stelle entlang der Kontur, um Haltestege hinzuzufügen.

Nachdem Sie einen Haltesteg hinzugefügt haben, können Sie dessen Position durch Klicken und Ziehen des Haltestegs anpassen:

Um einen manuell platzierten Haltesteg zu löschen, drücken Sie im Ansichtsbereich auf dem Haltesteg STRG+Linksklick (Windows) oder BEFEHLSTASTE+Linksklick (Mac).

Tipp: Wenn Sie Haltestege erhalten möchten, die nur manuell angegeben werden, wählen Sie im Dropdown-Menü Haltesteg-Positionierung die Option Anzahl der Haltestege aus und setzen Haltestege pro Kontur auf 0.
