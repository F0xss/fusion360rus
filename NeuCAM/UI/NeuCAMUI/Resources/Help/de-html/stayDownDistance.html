Bei vielen Werkzeugwegen wird ein Schnitt erstellt, dann erfolgt eine Neupositionierung für den nächsten Schnitt. Wenn der Abstand zwischen den Positionen größer ist als der Wert für maximalen Flächenkontakt, wird das Werkzeug zwischen den Schnitten zurückgezogen. Um das Werkzeug zwischen Schnitten nicht zurückzuziehen, legen Sie diese Option auf einen Wert fest, der über dem längsten Neupositionierungsabstand liegt.

Im dargestellten Bauteil beträgt die Breite an der breitesten Stelle 1.8 Zoll.

1" Maximaler Flächenkontakt-
das Werkzeug wird zurückgezogen

2" Maximaler Flächenkontakt-
das Werkzeug bleibt im Flächenkontakt

Veranschaulicht anhand von adaptiver 2D-Abtragung

Anmerkung: Unabhängig vom Flächenkontaktabstand zieht sich das Werkzeug zwischen den Durchgängen zurück, wenn die Eilgangbewegung das Werkstück beeinträchtigt.
