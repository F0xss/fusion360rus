Steuert, welches Koordinatensystem diesen Antastvorgang steuert.

Wenn diese Option ausgewählt ist, wird der Taster von einem anderen Koordinatensystem (WKS-Überschreibung) als dem im Setup definierten Koordinatensystem (WKS-Versatz) gesteuert. Die WKS-Überschreibung für diese Antastoperation wird nicht aktualisiert. Der WKS-Versatz für nachfolgende Bearbeitungsoperationen wird aktualisiert. Dies sperrt das Koordinatensystem für den Taster.

Beispiel für einen aktualisierten WKS-Versatz (1=G54) ohne geänderte WKS-Überschreibung (2=G55) für mehrere Bauteile:

Setup-WKS überschreiben ausgewählt

Wenn diese Option abgewählt ist, wird der Taster vom selben Koordinatensystem gesteuert wie das im Setup definierte (WKS-Versatz). Der WKS-Versatz für diese Antastoperation und die nachfolgenden Bearbeitungsoperationen werden aktualisiert.
