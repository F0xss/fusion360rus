Definiert die Flächen, Körper oder Komponenten, die nicht gestützt werden sollen.

Klicken Sie auf die einzelnen Flächen eines Körpers. Durch Doppelklicken wird der gesamte Körper ausgewählt. Durch Festlegen von vermiedenen Modellen können zwei verschiedene Auflager-Aktionen für denselben Körper erstellt werden. Das vermiedene Modell für dieses Auflager kann dann beispielsweise das gestützte Modell für ein anderes Auflager sein.
