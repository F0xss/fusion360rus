Wählen Sie eine interne oder externe kreisförmige Fläche aus. Diese kann eine gerade Wand umfassen oder verjüngt sein. Die Höhen- und Tiefeneinstellungen werden automatisch von der ausgewählten Fläche abgeleitet.

Bohrfräsen - Geometrieauswahl

Wählen Sie eine zylindrische oder konische Fläche aus dem Modell aus. Dabei kann es sich um eine Insel zum AD-Gewindefräsen oder eine Bohrung zum ID-Gewindefräsen handeln. Der Durchmesser, die Anfangshöhe und die Tiefe unten werden von dieser Auswahl abgeleitet.
