Der für den Ordner zu verwendende Mustertyp.

Linear - Ordnet Operationen/Werkzeugwege in einer geraden Linie bzw. in einem Raster an.

Kreisförmig - Ordnet Operationen/Werkzeugwege um einen Drehmittelpunkt an.

Spiegel - Erstellt Spiegelbild-Operationen/-Werkzeugwege, die an einer Spiegelebene umgekehrt werden.

Duplizierung - Erstellt eine Anordnung unter Verwendung von Operationen/Werkzeugwegen als Quelle und Skizzierpunkten als Ziel.

Komponente - Verfügbar für Nur Baugruppen. Diese Option ordnet Operationen/Werkzeugwege von einer Komponente für mehrere Komponenten in einer Baugruppe an.
