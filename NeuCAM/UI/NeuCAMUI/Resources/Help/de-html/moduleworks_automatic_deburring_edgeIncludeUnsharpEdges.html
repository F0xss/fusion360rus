Includes edges that need deburring despite being modeled as small fillets.

The size of the fillets that this option detects is affected by the Edge Width, Edge Height or Edge Radius values from the Passes tab.

Abgerundete Kante

Deaktiviert
Vermeidet abgerundete Kanten

Aktiviert
Bearbeitet abgerundete Kanten

Anmerkung: Das Einschließen von abgerundeten Kanten kann die Berechnungszeit erheblich verlängern.
