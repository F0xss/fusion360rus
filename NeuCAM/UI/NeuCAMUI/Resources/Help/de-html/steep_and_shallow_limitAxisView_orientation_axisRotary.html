Wählen Sie eine Ebene aus, zu der die Rotationsachse lotrecht ausgerichtet werden soll. Sie können auch eine Kante auswählen, zu der die Rotationsachse parallel platziert wird. Denken Sie daran, dass die Werkzeugachse immer lotrecht zur Rotationsachse verläuft.

Im unten stehenden Beispiel wird die Auswahl einer Ebene gezeigt, zu der die Rotationsachse lotrecht ausgerichtet werden soll. Außerdem wird der resultierende Werkzeugweg dargestellt, bei dem die Werkzeugachse immer parallel zur Ebene liegt. Die Werkzeugachse kann sich nicht in die Ebene hinein oder aus der Ebene heraus neigen.
