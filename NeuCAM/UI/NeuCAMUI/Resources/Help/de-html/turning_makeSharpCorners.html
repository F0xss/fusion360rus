Erzwingt eine scharfe Werkzeugwegüberschneidung an allen äußeren Ecken.

Wenn diese Option deaktiviert ist, verläuft der Werkzeugweg um alle scharfen äußeren Ecken. Damit wird ein Werkzeugweg erstellt, bei dem plötzliche Richtungsänderungen vermieden werden.

Deaktiviert - Gerollte Ecke

Aktiviert - Scharfe Ecke

Anmerkung: Dieser Parameter wirkt sich nur auf den Werkzeugweg aus. Das Bauteil besitzt immer eine scharfe Überschneidung. Durch Verwendung eines Werkzeugs mit kleinerem Spitzenradius kann es zu Bauteilkollisionen kommen.
