Gibt an, ob der Volumenanteil für homogenisierte Stützstrukturen manuell oder automatisch festgelegt wird.

Wählen Sie diese Option aus, um einen automatischen Volumenanteil zu verwenden.

Deaktivieren Sie diese Option, um manuell einen Volumenanteil festzulegen.

Gitterartige Stützstrukturen werden homogenisiert, anstatt direkt mit einem Netz versehen. Mit der Homogenisierung wird eine volumetrische Darstellung erstellt, wobei die Materialeigenschaften um einen automatisch berechneten Volumenanteil skaliert werden. Der Volumenanteil ist das Verhältnis des tatsächlichen Stützvolumens zum homogenisierten Volumen.
