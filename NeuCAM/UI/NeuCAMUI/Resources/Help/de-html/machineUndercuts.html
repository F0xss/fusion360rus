Verwendet Konturdurchgänge, um Hinterschnittbereiche zu bearbeiten, die andernfalls aufgrund von überhängender Geometrie nicht erreichbar sind. Hinterschnitte können mithilfe von 3 oder 5 Achsen bearbeitet werden.

Bei der 3-Achsen-Bearbeitung können Sie Werkzeuge wie z. B. Lollipop-, Scheiben-, Tonnen- oder Schwalbenschwanzfräser verwenden, um Hinterschnittbereiche zu erreichen.

Für die 5-Achs-Bearbeitung können Sie eine größere Palette von Werkzeugen verwenden, einschließlich eines Kugelfräsers. Zusätzlich zur Auswahl von Hinterschnitte bearbeiten müssen Sie auch die Einstellungen auf der Registerkarte Multi-Achsen verwenden, um einen 5-Achsen-Werkzeugweg zu generieren.

Im Allgemeinen können Sie durch die Verwendung von 5-Achsen-Bewegungen bei der Bearbeitung von Hinterschnitten mehr Bereiche in einer einzigen Operation erreichen, was zu Einsparungen bei der Programmierzeit führen kann.
