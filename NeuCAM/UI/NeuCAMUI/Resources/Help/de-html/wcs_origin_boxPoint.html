Wenn entweder die Ursprungsoption Punkt auf Rohteilrahmen oder die Ursprungsoption Punkt auf Modellrahmen ausgewählt wurde, können Sie das Dropdown-Menü Punkt auf virtuellem Rahmen verwenden, um einen Punkt (Beispiel: Mitte oben) auf dem virtuellen Rahmen zur Lokalisierung des WKS-Ursprungs anzugeben.
