Legt fest, wo Stabauflager erzeugt werden.

Mit Scharf werden Auflager an Ecken (Scheitelpunkten) hinzugefügt. Mit Per Cluster-Erkennung abstumpfen werden Auflager in kritischen Bereichen in der Mitte von Clustern hinzugefügt. Mit Scharfer Knoten außerhalb des Clusters werden Auflager an Ecken (Scheitelpunkten) hinzugefügt, die sich nicht innerhalb von Clustern befinden. Ein Cluster ist ein zusammenhängender Bereich, der gemäß Definition des Auflagerüberhangwinkels als Auflager benötigend gekennzeichnet ist.
