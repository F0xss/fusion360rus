Legt die Drehzahl der Spindel fest, gemessen in Umdrehungen pro Minute (U/min). Die Spindeldrehzahl bleibt während der Operation konstant.

Die Spindeldrehzahl ist eine Drehbewegung

Das Schneiden mit einer Spindeldrehzahl, die mithilfe der Empfehlungen des Werkzeugherstellers festgelegt wird, hilft Ihnen dabei, eine gute Oberflächengüte für Bauteile sicherzustellen. Sie können manuell eine Spindeldrehzahl eingeben oder die Spindeldrehzahl von Fusion 360 berechnen lassen, wenn Sie eine empfohlene Schnittgeschwindigkeit eingeben.

Anmerkung: Sie sollten die Spindeldrehzahl möglicherweise reduzieren, je nachdem, ob Sie eine Maschine mit geringer Leistung, wenig Kühlmittel oder Spannvorrichtungen mit niedrigem Klemmdruck verwenden.
