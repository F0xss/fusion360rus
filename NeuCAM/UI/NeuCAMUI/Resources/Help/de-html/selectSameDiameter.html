Wählt alle Bohrungen mit dem gleichen Durchmesser wie das aktuell ausgewählte Element aus. Wenn Sie z. B. eine einzelne 6-mm-Bohrung und eine einzelne 12-mm-Bohrung auswählen, wird automatisch jede 6-mm- und 12-mm-Bohrung auf dem Bauteil ausgewählt.

Eine einzelne Auswahl kann passende Bohrungen erkennen.

Anmerkung: Diese Option ist assoziativ zum Modell. Wenn zusätzliche Bohrungen mit demselben Durchmesser zum Modell hinzugefügt werden, werden die neuen Bohrungen bei der Neugenerierung der Operation automatisch eingeschlossen.
