Wählen Sie ein Werkzeug zum Stutzen aus:

Polygon - Ermöglicht das Platzieren von Punkten zum Erstellen eines Polygons über einen Werkzeugwegbereich.
Ebene - Ermöglicht das Angeben einer Ebene, um einen Bereich eines Werkzeugwegs abzudecken.
Begrenzung - Ermöglicht die Auswahl einer geschlossenen Kontur auf dem Modell, die als Begrenzung über einen Werkzeugwegbereich verwendet werden soll.

Ebenen und Begrenzungen sind assoziativ zum Modell, d. h., sie können neu generiert werden, wenn sich das Modell ändert.
