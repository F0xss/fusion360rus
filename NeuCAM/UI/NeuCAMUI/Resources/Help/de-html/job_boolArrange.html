Ordnet mehrere Komponenten automatisch so an, dass sie in das Grundvolumen der additiven Maschine passen.

Wenn Sie das Setup erstellt haben, wählen Sie die anzuordnenden Komponenten aus. Je nach Typ der im Setup ausgewählten additiven Maschine können Sie die Komponenten automatisch in 2D oder 3D anordnen. Komponenten, die nicht automatisch angeordnet werden können, werden außerhalb des Grundvolumens belassen und rot markiert.
