Ermöglicht Ihnen, die Verdickung, die auf Stützstrukturen angewendet wurde, in die 3MF-Datei für die Prozesssimulation einzuschließen.

Dies gilt nur für Stützstrukturen, die mit aktivierter Option Strukturen verdicken erstellt werden. Wenn Sie das Kontrollkästchen deaktivieren, wird ein vorverdicktes Ersatz-Auflagernetz für die Prozesssimulation verwendet.
