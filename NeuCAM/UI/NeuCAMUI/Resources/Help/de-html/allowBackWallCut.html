Ermöglicht es, dass Ein-/Ausfahrradius/-länge/-winkel über die Grenzen der Schnittbegrenzung hinaus bewegt werden können. Dies kann dazu führen, dass der Werkzeugweg mit dem übrigen Rohteil kollidiert.

Wenn diese Option deaktiviert ist, wird der Werkzeugweg nicht generiert, wenn er durch das verbleibende Rohteil verläuft.
