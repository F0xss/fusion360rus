Diese Option wird in Orange dargestellt und legt die Referenzposition für den Abstandsradius fest. Die Referenz kann in Bezug zum Rohteil, zum Modell, einem angegebenen Radius, Durchmesser oder einer der anderen radialen Positionen stehen. Diese Referenzposition kann mit einem positiven oder negativen Versatzwert verschoben werden.

Rückzug - Legt den Abstandsradius in Bezug auf die Rückzugsposition fest.

Rohteil-OD - Legt den Abstandsradius in Bezug auf den Außendurchmesser des definierten Rohteils fest.

Modell-OD - Legt den Abstandsradius in Bezug auf den Außendurchmesser des definierten Modells fest.

Außenradius - Legt den Abstandsradius in Bezug auf die Position des Außenradius fest. Der Abstandsradius muss größer sein als der Außenradius. Verwenden Sie den Parameter Versatz, um bei Bedarf Anpassungen vorzunehmen.

Innenradius - Legt den Abstandsradius in Bezug auf die Position des Innenradius fest. Der Abstandsradius muss größer sein als der Innenradius. Verwenden Sie den Parameter Versatz, um bei Bedarf Anpassungen vorzunehmen.

Modell-ID - Legt den Abstandsradius in Bezug auf den Innendurchmesser des Modells gemäß Definition im Setup fest. Verwenden Sie den Parameter Versatz, um bei Bedarf Anpassungen vorzunehmen.

Rohteil-ID - Legt den Abstandsradius in Bezug auf den Innendurchmesser des Rohteils gemäß Definition im Setup fest. Verwenden Sie den Parameter Versatz, um bei Bedarf Anpassungen vorzunehmen.

Auswahl - Wählen Sie eine Fläche, einen Scheitelpunkt oder einen Punkt im Modell zur Definition des Abstandsradius aus. Verwenden Sie den Parameter Versatz, um nach Bedarf Anpassungen mit positiven oder negativen Werten vorzunehmen.

Radius - Mit dieser Option können Sie einen Wert für den Radius in das Feld Versatz eingeben. Dieser Wert wird in Bezug auf die Mittellinie des Bauteils festgelegt, und es werden keine assoziativen Änderungen am Modell erkannt.

Durchmesser - Mit dieser Option können Sie einen Wert für den Durchmesser in das Feld Versatz eingeben. Dieser Wert wird in Bezug auf die Mittellinie des Bauteils festgelegt, und es werden keine assoziativen Änderungen am Modell erkannt.

Anmerkung: Der Abstandsradius muss gleich oder größer als der Rückzugs-, Außen- und Innenradius sein, um einen gültigen Werkzeugweg zu generieren.
