Gibt den Typ der Durchgänge an, die für steile Bereiche verwendet werden sollen.

Entlang - Durchgänge verlaufen entlang der Richtung des Restmaterials. Wenn Sie beispielsweise eine Innenecke bearbeiten, verlaufen die Durchgänge entlang der Innenecke.

  • Mit maximaler Querzustellung - Die Querzustellung wird so angepasst, dass sie immer innerhalb des Restmaterialbereichs bleibt, sodass die Oberflächengüte des umgebenden Bereichs nicht beeinflusst wird. Die maximale Querzustellung tritt am breitesten Teil des Restmaterials auf. Wenn das Restmaterial schmaler wird, bewegen sich die Querzustellungen näher zueinander, sodass weniger Material zum Abtragen für die einzelnen Querzustellungen übrig bleibt, was möglicherweise dazu führt, dass das Werkzeug auf dem Bauteil scheuert.
  • Mit konstanter Querzustellung - Eine konstante Querzustellung wird im gesamten Restmaterialbereich beibehalten, sodass Sie eine konstante Spandicke erhalten. Dies bedeutet bessere Schnittbedingungen. Wenn das Restmaterial schmaler wird, können die äußeren Durchgänge den bereits bearbeiteten, umgebenden Bereich beeinträchtigen, wodurch möglicherweise Spuren auf der Oberfläche entstehen.

Über - Durchgänge verlaufen über die Richtung des Restmaterials. Wenn Sie beispielsweise eine Innenecke bearbeiten, verlaufen die Durchgänge über die Innenecke.
