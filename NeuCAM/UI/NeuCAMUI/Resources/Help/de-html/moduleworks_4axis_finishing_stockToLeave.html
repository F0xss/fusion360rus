Gibt das Rohteil an, das an Wänden und Böden beibehalten werden soll.

Die Angabe eines positiven Werts für Rohteil-Aufmaß führt dazu, dass zusätzliches Material auf dem Bauteil verbleibt. Bei Schruppoperationen bleibt häufig eine geringe Menge an Rohteilmaterial übrig, das dann mit zukünftigen Schlichtoperationen entfernt werden kann.
