Gibt den zulässigen Wert für die Winkelabweichung des geometrischen Elements an. Diese bilaterale Toleranz ist der Wert, um den der Winkel des geometrischen Elements variieren kann, sodass er trotzdem noch als innerhalb der Toleranz betrachtet wird.

Ein geometrisches Element, dessen Abweichung über diesem Wert liegt, wird als außerhalb der Toleranz betrachtet. Wenn Sie beispielsweise eine Fläche mithilfe von Winkel entlang der Y-Achse antasten, wird der Winkel von der Fläche zur Ebene gemessen, die lotrecht zur Y-Achse liegt.

1 - Ebene lotrecht zur Y-Achse.
