Steuert, ob die Werkzeugwege im Setup auf einen einzelnen Nullpunkt oder auf mehrere Nullpunkte auf der Maschine angewendet werden.

Wählen Sie diese Option aus, um alle Werkzeugwege im Setup in mehrere Versätze zu duplizieren, die auf der Maschine eingerichtet sind. Dies ist hilfreich, wenn Sie identische Jobs zum Bearbeiten haben. Es werden zusätzliche Einstellungen angezeigt, um die Anzahl der Instanzen anzugeben, die Sie bearbeiten möchten.
