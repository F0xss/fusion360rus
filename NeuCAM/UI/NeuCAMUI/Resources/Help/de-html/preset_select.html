Enthält alle vordefinierten Schnittdaten, die Sie erstellt haben. Dies ist eine Möglichkeit, die Parameter Spindeldrehzahl und Vorschub basierend auf Ihren bevorzugten Einstellungen schnell auszufüllen. Diese Schnittdaten-Voreinstellungen werden mit dem aktuellen Werkzeug gespeichert.

Wählen Sie eine Voreinstellung aus der Liste im Pulldown-Menü aus.

So fügen Sie eine Voreinstellung hinzu:

  • Klicken Sie auf Auswählen, um die Werkzeugbibliothek aufzurufen.
  • Wählen Sie das Symbol Werkzeug bearbeiten aus.
  • Klicken Sie auf der Registerkarte Schnittdaten auf das Symbol.
  • Geben Sie diesem einen beschreibenden Namen, und geben Sie Ihre bevorzugten Drehzahlen und Vorschübe ein.
