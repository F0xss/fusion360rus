Gibt die Überprüfungsergebnisse an, die zum Berechnen der Bauteilausrichtung verwendet werden.

Wenn Sie bereits Überprüfungsergebnisse importiert haben und diese zur Berechnung der Bauteilausrichtung verwenden möchten, wählen Sie die Überprüfungsergebnisse manuell im Browser aus.

Andernfalls wird die Bauteilausrichtung berechnet, wenn Sie auf Ergebnisse abrufen > Überprüfungsergebnisse importieren klicken und eine maschinengenerierte Ergebnisdatei wählen.
