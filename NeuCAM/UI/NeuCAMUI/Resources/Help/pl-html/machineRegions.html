Określa, czy model jest traktowany jako posiadający tylko obszary o małym nachyleniu, tylko obszary o dużym nachyleniu, czy jako połączenie obu tych obszarów.

Obszar o dużym i małym nachyleniu — rozpoznaje cały model jako zawierający obszary o dużym nachyleniu i obszary o małym nachyleniu. Kąt progu określa, które obszary są strome i które są płytkie.

Tylko o małym nachyleniu — rozpoznaje cały model jako zawierający obszary o małym nachyleniu, co umożliwia utworzenie ścieżki narzędzia Równoległy lub Wytłaczany oraz uzyskanie dostępu do dodatkowych funkcji, takich jak możliwość optymalizacji przejść prostopadłych.

Tylko o dużym nachyleniu — rozpoznaje cały model jako zawierający obszary o dużym nachyleniu, co umożliwia utworzenie ścieżki narzędzia Kontur i uzyskanie dostępu do dodatkowych funkcji, takich jak możliwość obróbki podcięć.

Stromy i płytki

Tylko o małym nachyleniu
Przejścia zoptymalizowane

Tylko o dużym nachyleniu
Obróbka podcięć

Wskazówka: można wygenerować ścieżkę narzędzia na całym modelu lub użyć opcje zawierania Granica obróbki lub Nachylenie na karcie Geometria, aby określić obszar docelowy.
