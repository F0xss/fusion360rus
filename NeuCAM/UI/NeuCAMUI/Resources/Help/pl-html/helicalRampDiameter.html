Maksymalna średnica dla wejścia helisy do stempla.

Optymalna wartość powoduje, że narzędzie nakłada się na środek, podczas gdy nadal tworzony jest maksymalny otwór śrubowy dla wejścia do wnęki. Celem jest dobre usuwanie wiórów. Jeśli wartość jest większa niż średnica narzędzia, może pozostawić kominek stojący w środku linii śrubowej.

Wartość 1,8 x śred.

Wartość 0,8 x śred.

System określi najlepsze położenie linii helisy wejścia, chyba że zostanie wybrana opcja Położenie wejścia. (Położenia wjazdu pokazane poniżej). Cały obszar będzie zawsze usunięty, niezależnie od średnicy. Uwaga: szerokość obszaru powinna być równa średnicy narzędzia + średnicy zejścia + przygotówka, w przeciwnym razie wejście helikalne nie powiedzie się.
