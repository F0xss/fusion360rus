Określa długość krawędzi wokseli lub elementów, które są używane do reprezentowania komponentów podczas obliczania układu.

Im mniejszy rozmiar woksela, tym dokładniej reprezentowany jest kształt każdego komponentu, co zapewnia lepsze wykorzystanie dostępnego miejsca.

Uwaga: obliczenie układu przy użyciu mniejszego rozmiaru woksela może potrwać dłużej.
