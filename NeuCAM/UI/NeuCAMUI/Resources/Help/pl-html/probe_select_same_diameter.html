Wybiera wszystkie otwory i kominki o tej samej średnicy co aktualnie wybrany otwór lub kominek. Na przykład wybranie pojedynczego otworu o średnicy 15 mm automatycznie wybiera trzy inne otwory o tej samej średnicy.

Jeśli otwory lub kominki o różnych średnicach zostaną wybrane wielokrotnie, wszystkie otwory lub kominki o tych średnicach zostaną wybrane. Jeśli otwory lub kominki o tej samej średnicy zostaną dodane później, ponowne wygenerowanie operacji automatycznie spowoduje uwzględnienie w cyklu probingu dodanych otworów lub kominków.
