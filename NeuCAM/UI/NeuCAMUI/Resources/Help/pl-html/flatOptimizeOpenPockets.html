Zapewnia narzędziu większą swobodę podejścia do części z otwartych stron kieszeni. Powoduje to zwiększenie wydajności ruchów wejściowych, takich jak krótkie, styczne przedłużenie zamiast ruchu rampowego, oraz lepsze warunki skrawania, takie jak:

  • Redukowanie skrawań, które mają miejsce przy pełnej szerokości narzędzia.
  • Redukowanie ostrych naroży w ścieżce narzędzia.

Gdy opcja jest nie jest zaznaczona, narzędzie nie może wchodzić z zewnątrz części, co jest przydatne, gdy chcesz uniknąć zacisków lub mocowań, które utrudniają obróbkę. Następuje również mniej podniesień narzędzia, co przyspiesza obróbkę bardziej miękkich materiałów, takich jak pianka, gdzie warunki skrawania nie są tak krytyczne.
