Ustawia prędkość wrzeciona podczas wykonywania ruchu zejścia, który jest ruchem umożliwiającym narzędziu stopniowe wchodzenie w blok materiału.

Wrzeciono musi działać z niższymi prędkościami podczas wykonywania ruchu zejścia. Jest to spowodowane tym, że ruch zejścia zbiera więcej materiału niż normalny ruch skrawający.

Uwaga: obroty wrzeciona podczas zagłębiania mogą nie być obsługiwana przez wszystkie postprocesory.
