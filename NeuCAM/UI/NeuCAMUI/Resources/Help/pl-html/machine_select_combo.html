Menu rozwijane Maszyna umożliwia wybranie wcześniej używanych maszyn lub wybór opcji Brak, aby usunąć wybraną maszynę. Wybranie typu maszyny określa, które strategie będą dostępne.

Naciśnij Wybierz, aby wyświetlić dostępne Typy obrabiarki. Wskazanie procesu obróbki filtruje listę maszyn.

Wybór typu obrabiarki określa, które strategie będą dostępne.

Frezowanie — maszyny z wrzecionami obrotowymi i 2–5 osiami ruchu.

Toczenie — 2-osiowe tokarki i tokarki z konfiguracjami frezowania/skrętu.

Cięcie — urządzenia wykorzystujące strumień wody, laserowe lub plazmowe.

Wytwarzanie przyrostowe — drukowanie 3D z użyciem tworzyw sztucznych lub metalu.
