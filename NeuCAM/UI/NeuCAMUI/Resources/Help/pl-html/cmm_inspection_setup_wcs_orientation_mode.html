Menu rozwijane Orientacja zawiera następujące opcje definiujące osie X, Y i Z.

Orientacja modelu — używa orientacji bieżącej części dla orientacji RUW.

Wybierz oś/płaszczyznę Z i oś X — wybierz powierzchnię lub krawędź, aby zdefiniować oś Z i inną powierzchnię lub krawędź, aby zdefiniować oś X. Osie Z i X mogą być odwrócone o 180 stopni.

Wybierz oś/płaszczyznę Z i oś Y — wybierz powierzchnię lub krawędź, aby zdefiniować oś Z i inną powierzchnię lub krawędź, aby zdefiniować oś Y. Osie Z i Y mogą być odwrócone o 180 stopni.

Wybierz osie X i Y — wybierz powierzchnię lub krawędź, aby zdefiniować oś X i inną powierzchnię lub krawędź, aby zdefiniować oś Y. Osie X i Y mogą być odwrócone o 180 stopni.

Wybierz układ współrzędnych — ustawia orientację RUW dla ustawienia ze zdefiniowanego lokalnego układu współrzędnych (LUW) w modelu. Wykorzystuje zarówno początek, jak i orientację istniejącego układu współrzędnych. Użyj tej opcji, jeśli model nie zawiera odpowiedniego punktu i płaszczyzny dla operacji.
