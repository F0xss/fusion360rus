Specifies which spindle chuck to open before returning the secondary spindle.

Keep Both Clamped - If main spindle has bar loaded and sub spindle has the workpiece, both spindles are kept clamped to continue machining.

Unclamp Primary Spindle - if the workpiece is a billet and the main and sub spindle are clamped on the part, the main spindle will need to be unclamped so the sub spindle can move with the part.

Unclamp Secondary Spindle - If the sub spindle has been used for bar pulling or supporting the workpiece, the sub spindle can now return home without the workpiece.
