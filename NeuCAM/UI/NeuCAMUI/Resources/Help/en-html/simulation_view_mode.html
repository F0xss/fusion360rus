Controls how you view the simulation.

Machine base - Maintains the view such that the base of the machine appears stationary on-screen while the rotary and linear axes, the tool, and the model move relative to it.
Model - Maintains the view such that the model appears stationary on-screen while the tool moves relative to it.
Tool - Maintains the view such that the tool appears stationary on-screen while the model moves relative to it. Use this option when zooming in to view the movement of the tool more closely. Also useful for visualizing threading operations or rotary operations.
