Diameter Range - Size based selection.

The system will evaluate the model based on the Minimum and Maximum Diameter values. Use this range to include or exclude hole sizes.
Example: Select all .250 - .2501 diameter holes for drilling and all .2505 - .2506 diameter holes for reaming.

Using Model based selections insures associativity to any future changes of the feature.

Note: Spot Drilling Options
* If you select the modeled chamfered face of a hole. The Bottom Height (depth) will automatically be determined to the theoretical tip.
* If you select by Diameter Range to include all holes. Set the Bottom Height from Chamfer Width and set the Chamfer Width value.
