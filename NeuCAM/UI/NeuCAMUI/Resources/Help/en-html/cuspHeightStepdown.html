Specifies the desired cusp height to calculate the stepover, stepdown, or both.

Cusps are excess material that run along a toolpath and are a result of the gap between the position of the tool in successive passes. Generally, the smaller the stepdown and stepover, the smaller the cusp height.

In reverse, you can use a desired cusp height to calculate the stepover and stepdown value for a toolpath. Using the cusp height helps give an even surface finish across the part, independent of the tool selection.

1 - Cusp Height
2 - Stepdown
3 - Stepover
