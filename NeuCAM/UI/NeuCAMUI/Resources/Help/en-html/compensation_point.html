The compensation point is used to drive the tool along the contours, which are selected during toolpath creation. The Sideways Compensation parameter of a toolpath specifies which side of the tool the compensation point sits on.

If no compensation point is selected, Fusion 360 automatically selects one. Vertically, the compensation point is level with the tip of the tool (Z0 location). Horizontally, the compensation point aligns with the maximum diameter of the tool. In the image below, the compensation point is represented by the red dot.
