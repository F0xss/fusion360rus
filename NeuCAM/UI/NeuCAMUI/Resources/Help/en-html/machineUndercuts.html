Uses contour passes to machine undercut areas that are otherwise unreachable, because of overhanging geometry. Undercuts can be machined using 3 axes or 5 axes.

For 3-axis machining, you can use tools such as a lollipop, disc, barrel, or dovetail tool to reach undercut areas.

For 5-axis machining, you can use a wider range of tools, including a ball-nosed tool. In addition to selecting Machine Undercuts, you also need to use the settings in the Multi-Axis tab to generate a 5-axis toolpath.

Generally, using 5-axis motion when machining undercuts gives you the ability to reach more areas in a single operation, which can save on programming time.
