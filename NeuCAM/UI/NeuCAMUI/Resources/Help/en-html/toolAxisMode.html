Specifies the primary way that the tool should tilt when machining the part.

Vertical - Aligns the tool with the specified tool orientation.

From Point - The tip of the tool points away from a sketched point.

To Point - The tip of the tool points towards a sketched point.

From Curve - The tip of the tool points away from a sketched curve.

To Curve - The tip of the tool points towards a sketched curve.
