Specifies the type of connection move between passes that are nearby each other on the same radial plane.

  • No contact - Uses a retract move to connect the passes. The retraction distance may depend on the Retraction Policy.
  • Straight line - Uses a straight line to connect the passes.
  • Smooth - Uses smooth tangential movements to connect the passes.
