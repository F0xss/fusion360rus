Specifies what to do when the tool encounters inner corners that it cannot reach because of the geometry of the tool.

Do not deburr - Moves the tool to the next edge of the inner corner after it can no longer machine any more of the current edge. As a result, burrs remain on some inner corners. Use this option to prioritise a faster cycle time over deburring more of the internal corners.

Unreachable corner

Burrs on inner corners

Cut with tip of tool - Attempts to cut more of the inner corner by using more of the tip of the tool. This removes as much material as possible from inner corners.

Cutting with tip of tool

Inner corners deburred

Note: To use the Cut at Fixed Point on Tool feature, you need to select Do not deburr.
