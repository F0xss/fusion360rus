Specifies how the workpiece coordinate system is specified.

Origin & Orientation

Coordinate System

Stock & Orientation

Use Origin & Orientation - A sketch point or stock vertex is used to define the location of the WCS, and a plane or planar face is used to define the orientation of the WCS.

Use Coordinate System - An existing user coordinate system (UCS) is used to define both the location and orientation of the WCS.

Use Stock & Orientation - A location on the stock (Example: Top Center) is used to define the location of the WCS, and a plane or planar face is used to define the orientation of the WCS.
