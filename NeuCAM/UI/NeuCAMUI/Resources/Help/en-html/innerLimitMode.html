Limits the X axis position in reference to Radii Limits or the Contact point of the tool nose radius.

The dark blue line represents the Inner Radius limit.

Cutting Edge sets a hard limit for the X position of the cut. This limit is determined by the Inner Radius, for OD machining, or the Outer Radius, for ID machining.

Cutting Edge limit

Contact Point positions the nose radius of the tool to the tangent point of the cut. A larger nose radius will create a greater overlap.
