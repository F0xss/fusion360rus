Specifies the primary mode of the tool axis. Depending on the model's geometry, select the option that allows you to reach the most areas with the tool.

Primary Mode for 4-axis machining:

  • To rotary axis - The tip of the tool always points to the rotary axis.
  • To rotary axis with tilt - Firstly, the tip of the tool points to the rotary axis. And then, the tool tilts towards the rotary axis.
  • Zero lead angle with tilt - Firstly, the tool aligns with the average angle between the two edges. And then, the tool tilts towards the rotary axis.

Primary Mode for 5-axis machining:

  • Lead - Tilts the tool in the direction of travel.
  • Tilt to Z - Tilts the tool to any location on a conical boundary that is around the tool orientation Z axis.
