Preferred Angle for 4-axis machining:

Tilts the tool towards the rotary axis by an angle, which is measured from the tool orientation Z axis.

Preferred Angle for 5-axis machining:

Tilts the tool to the tool orientation Z axis by the specified angle. A Preferred Tilt Angle of 0 aligns the tool with the tool orientation Z axis.

This angle can be thought of as a conical boundary around the tool orientation Z axis.
