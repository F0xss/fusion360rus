Opens the Tool Library, where you can select a nozzle for your work type.

Select the appropriate Tool/Nozzle for your cutting process.

Appropriate Tool/Nozzle Types
Waterjet cutting
Laser cutting
Plasma cutting

You can also select the
New Waterjet/Plasma/Laser
icon to create a custom Tool/Nozzle.

Note: The type of Tool/Nozzle you select will effect some of the toolpath options.
