Controls which coordinate system drives this probing operation.

When selected, the probe is driven from a different coordinate system (WCS Override) to that defined in the setup (WCS Offset). The WCS Override for this probing operation is not updated, but the WCS Offset for subsequent machining operations is updated. This locks the coordinate system for the probe.

When deselected, the probe is driven from the same coordinate system as that defined in the setup (WCS Offset). The WCS Offset for this probing operation and subsequent machining operations is updated.
