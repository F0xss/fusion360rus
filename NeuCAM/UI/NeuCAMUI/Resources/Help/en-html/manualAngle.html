Specifies the direction for all passes.

A manual pass direction of 0 degrees runs parallel to the x-axis of the WCS. A pass direction of 90 degrees runs parallel to the y-axis of WCS.

An example of when you might use a manual pass direction instead of an automatic direction is when the automatic direction results in passes that run up to a wall, causing the tool to touch the wall multiple times and leaving undesired marks.
