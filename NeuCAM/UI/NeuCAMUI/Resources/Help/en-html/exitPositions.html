Determines the exit position of the tool for safer machining.

Select a point to determine the exit position. The point must be as close as possible to the area you are machining, like the entrance of an open pocket for example, for the point to be detected during toolpath calculation.

If you select more than one point, the tool exits from the point that it is closest to after it has completed performing the cutting moves.

In the example below, an open pocket with an undercut is being machined. Initially, the tool retracts at the end of the pocket, which causes a collision with the geometry directly above it. Using an exit position, you can force the tool to stay down and exit the pocket so it can retract with sufficient clearance.

Available for 2D Pocket, 2D Contour, and 2D Adaptive.

Note: The tool performs a stay-down link move when moving to an exit position. The Maximum Stay-Down Distance must be large enough to accommodate for this move. for 2D Pocket and 2D Contour, you need to enable the Keep Tool Down check box to enter a Maximum Stay-Down Distance.
