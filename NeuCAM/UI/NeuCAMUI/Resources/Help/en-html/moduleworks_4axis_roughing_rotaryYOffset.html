Offsets the tool centerline to help cut with the edges of the tool rather than its center.

Tool Diameter = 0.3 in
Tool Offset = 0.14 in

The tool offset is applied in the opposite direction to the rotation direction of the rotary axis. When the rotation direction changes, to help prevent any damage, the tool automatically changes position by performing a linear movement without rotary movement.
