Form Mill allows you to create a custom shaped tool by using an imported cad drawing or a sketch. Most tooling manufacturers have CAD drawings available on their web sites.

The boundary should be 2D, flat and closed. Broken lines, like construction lines, are not allowed.

Once created the tool will be saved to the current Document in the Tool Library. You can open the Tool Library to add additional cutting parameters or descriptions. Move the tool to any Local or Cloud library for future use.
