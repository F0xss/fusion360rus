Sets the direction in which the tool should tilt when avoiding a collision.

Automatic: Automatically selects the direction in which to tilt the tool away from the part until the collision is avoided. Useful for parts that contain lots of features, but it may increase the time needed to calculate the toolpath. Available only when Vertical is selected as the Primary Mode for the tool axis.

For simpler parts, the following To and From options can provide smoother tool axis movement and use less time to calculate the toolpath.

When avoiding a collision:

From point: Tilts the tip of the tool in a direction that is away from a point.
To point: Tilts the tip of the tool in a direction that is towards a point.
From curve: Tilts the tip of the tool in a direction that is away from a curve.
To curve: Tilts the tip of the tool in a direction that is towards a curve.

Note: If the tool needs to tilt beyond the Maximum Tilt tool axis limit to avoid a collision, then that portion of the toolpath is removed.
