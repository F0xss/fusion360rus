For tools with special geometry where chip thinning control is important. This activates additional controls to set the Entry Radius blend and Exit Distance.

Only available when the Cycle type is set to Back cutting.

C = Cut direction - Away from the Chuck
R = Radius for Back Cutting entry
D = Distance for feed reduction

Starting the toolpath closest to the chuck side, the Radius blends onto the cut and feeds in the positive direction. At a distance before the end of the cut, the feedrate is reduced to prevent chipping of the part or the tool.
