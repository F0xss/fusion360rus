Specifies how tabs are placed.

  • By Distance - Lets you specify a Tab Distance to place tabs at an equal distance around the profile. Using the Positioning Method drop-down menu, you can choose to place tabs along the contour or along segments of a toolpath.
  • Number of tabs - Lets you specify the Number of tabs to place at an equal distance around the profile.

Note: Tabs are placed perpendicular to the geometry.
