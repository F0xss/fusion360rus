Maintains the tool axis at the specified preferred angle.

The Preferred Angle is measured from the specified tool orientation (if you do not specify a tool orientation in the Geometry tab, then the Setup Z axis is where the Preferred Angle is measured from). Where possible, maintaining the tool axis at a constant angle provides a better surface finish and helps maintain the programmed feed rate.

The preferred angle can be thought of as a conical boundary around the specified tool orientation.

Preferred Angle measured from tool orientation Z axis

To determine where the tool sits on the conical boundary, non-vertical Primary Mode settings provide the tool with an initial tilting direction, as they would if you weren't using the Override Tilt Angle. From the initial tilting direction, the tool axis moves to align with the edge on the boundary that it is closest to.
