Determines how fast the tool moves along the material (feed) and how fast the tool rotates in the spindle (speed). These parameters are interrelated by various expressions.

Following the tool manufacturer's recommendations for feeds and speeds helps to remove material at an efficient rate and ensure a longer tool life. With the material information of the tool and the stock, consult the tool manufacturer's manual for the recommended surface speed and feed per tooth, which Fusion 360 uses to determine the feeds and speeds.

Tip: Some tool manufacturers provide the actual feeds and speeds, which you can enter directly.
