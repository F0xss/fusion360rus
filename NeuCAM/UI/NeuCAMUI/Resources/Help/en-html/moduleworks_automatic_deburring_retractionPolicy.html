Controls how the tool moves between cutting passes.

Full Retraction - Retracts the tool to the Clearance Area before moving to the next cutting move. This is the safest retraction option, however, it results in a larger retraction move.

Smooth Retraction - Retracts the tool in a smooth motion, which is a shorter retraction. The tool uses the Full Retraction option when the smooth retraction cannot be achieved safely.
