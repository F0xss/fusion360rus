Offsets the wrap radius value. Offsetting this value controls how much material is removed from the edge of the walls. Commonly used when the walls of the crylindrical pocket are not modeled accurately for wrap machining.

When performing the wrapping, the toolpath can only be accurate at one radius, the Wrap Radius, which defaults to the radius of the selected wrap cylinder.

An offset value of 0 uses the radius from the selected wrap cylinder. In the image below, the cylindrical surface at the bottom of the tool is selected. As a result, the tool stops at the end of the bottom edge.

In the image below, the wrap radius is offset by an inch. As a result, the tool moves further into the wall.
