Specifies that the contouring operation will be used to create a chamfer.

Geometry Selection Tips:

Sharp Corners

Chamfered Edges

Sharp Corners - Select the sharp corner and define the size of the chamfer using the Chamfer Width Setting.

Chamfered Edges - Select the bottom edge of the chamfer. The chamfer width is calculated automatically.

Important: All edges selected must be either the bottom edge of a chamfered face or unchamfered edges. If both edge types are selected, then the edges that already have the chamfers modeled end up with chamfers that are twice the size they should be.
