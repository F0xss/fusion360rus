Specifies the ordering of the individual operations.

Preserve order - Specifies that features are machined in the order in which they were selected.

Order by operation - Specifies the ordering of the individual operations.

Order by tool - Specifies the ordering of operations by tool. For example, all operations that use a 3/4" diameter flat mill are machined first. Then the next set of operations that use another identical tool, a 1/4" diameter flat mill for example, are machined next.
