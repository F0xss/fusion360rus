Selecting the Machine Type defines which strategies will be available.

Press Select to view the available Machine Types. Picking the machining process, filters the list of machines.

Milling - Machines with rotary spindles and 2 - 5 axis of motion.

Turning - 2 axis Lathes and Lathes with mill/turn configurations.

Cutting - Waterjet, Laser or Plasma machines.

Additive - 3D printing using plastics or metal
