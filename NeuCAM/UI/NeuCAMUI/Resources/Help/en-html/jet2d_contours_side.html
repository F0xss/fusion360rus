Used to offset selected edges or sketches

Use this to offset the contour inside or outside from the original edge or sketch geometry. Can be used to Etch along the edge of a profile. Allows you to shift from the selected Sideways Compensation direction or reverse the compensation offset direction.

* Start Outside
* Start Inside
* Always Outside
* Always Inside

Note - If the Tool - Cutting Mode is set to Etch, Sideways Compensation will default to Center. Changing the Side options will not work unless Sideways Compensation is set to Left or Right.
