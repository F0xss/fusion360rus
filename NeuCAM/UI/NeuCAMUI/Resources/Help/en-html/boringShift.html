Specifies an incremental distance to move the boring bar insert away from the wall, before the tool is retracted. This move is usually made in the X direction. This can be applied to boring bars and back face chamfering tools.

Some controllers will include a parameter to change angular orientation of the spindle, and position the insert tip to the 3 o'clock position.
