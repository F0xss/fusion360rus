Offsets the profile of the Reference Tool by the specified amount. This gives a larger reference tool, which detects more remaining stock.

When the reference tool's cutting geometry is the same size as a corner, the reference tool reaches all of the corner in some regions, and in other regions, it may not reach the entirety of the corner. This can result in fragmanted areas of toolpath throughout the corner. To avoid this, you can increase the size of the reference tool, which results in detecting more material and generating a more desirable toolpath.
