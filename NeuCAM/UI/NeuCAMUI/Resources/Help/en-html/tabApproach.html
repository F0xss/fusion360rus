Determines how to position the tabs on the toolpath.

Contour - Uses the entire length of the contour to place tabs within. Tabs are more likely to be evenly spaced when compared with the Segment option.

Contour

Evenly spaced tabs

Segment - Uses segments, which are areas of the toolpath between two points, to place the tabs within. If the Tab Distance is larger than the segment length, then the segment is ignored, and no tab is placed within the segment.

Segment

One segment used
