Sets the rotational speed of the spindle, typically given in revolutions per minute (RPM). The spindle speed is expressed as an S code in the NC code.

Drilling with a spindle speed that is determined by using the tool manufacturer's recommendations helps ensure a good surface finish for your holes. You can enter a spindle speed manually, or let Fusion 360 calculate the spindle speed when you enter a value for the Surface Speed.

Note: You may want to reduce the spindle speed depending on if you have a low-powered machine, lack of coolant, or clamping fixtures that have low clamping pressure.
