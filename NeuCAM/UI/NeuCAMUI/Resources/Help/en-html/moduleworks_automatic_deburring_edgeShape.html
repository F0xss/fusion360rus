Controls how much material is removed from an edge.

Chamfering:

  • Constant Width - The distance between the two edges of the chamfer remains the same for each edge.
  • Constant Height - The depth of the chamfer remains the same for each edge.

Filleting:

  • Constant Width - The distance between the two ends of the fillet remains the same for each edge.
  • Constant Radius - The radius of the fillet remains the same for each edge.
