Allows you to select the machining area by continous Faces, single Faces or Contour Pairs of chains.

Drive Mode = Surfaces
Selection Mode = Faces
Select continous surfaces.

Drive Mode = Surfaces
Selection Mode = Manual
Select the individual surfaces.

Drive Mode = Contours
Selection Mode = Contour Pairs
Select the Lower and Upper Contour
that will drive the Swarf Toolpath.
Select the Lower Chain First.

Drive Mode = Contours
Selection Mode = Manual
Select the individual Edges
that will drive the Swarf Toolpath.
Select the Lower Chain First.
