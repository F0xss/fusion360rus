Specifies how the tool orientation is specified.

  • Use WCS is the default, and uses the workpiece coordinate system (WCS) of the current job for the tool orientation.
  • Use Point & Plane sets a specific tool orientation for this operation from any selected point and plane as the origin and orientation, respectively.
  • Use Coordinate System sets a specific tool orientation for this operation from any coordinate system feature in the model. This uses both the origin and orientation of the coordinate system feature. Use this if your model does not contain a suitable point & plane for your operation.
