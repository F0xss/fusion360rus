Helps determine the positions where the tool should lead-in, which is the move before the beginning of a cutting move.

Select points on the model to determine the lead-in positions. You can select one position or multiple positions. If you select multiple positions, the position that is the closest to a contour gets used as the approximate lead-in position.

Note: The positions you specify are approximate and may not always be the location where the tool leads in.
