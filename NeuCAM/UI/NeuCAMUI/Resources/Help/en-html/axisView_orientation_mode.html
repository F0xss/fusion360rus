Specifies the orientation of the rotary axis to match how your part is set up on the machine.

Setup X Axis - Uses the X axis of the workpiece coordinate system (WCS) of the current setup as the rotary axis.

Setup Y Axis - Uses the Y axis of the workpiece coordinate system (WCS) of the current setup as the rotary axis.

Setup Z Axis - Uses the Z axis of the workpiece coordinate system (WCS) of the current setup as the rotary axis.

Model orientation - Uses the coordinate system (WCS) of the current part as the rotary axis.

Select rotary axis - Select a flat face to set the rotary axis perpendicular to, or select a circular face to set the center of the face as the point the rotary axis runs through.

Select coordinate system - Sets a specific rotary axis orientation for this operation from a defined User Coordinate System (UCS) in the model. This uses both the origin and orientation of the existing coordinate system. Use this option if your model does not contain a suitable point and plane for your operation.
