Specifies how to identify the unmachined regions of a part where there is still stock material to remove. The resulting toolpath targets only unmachined regions.

  • From previous operation(s) - Uses all toolpaths in the current setup to identify remaining unmachined regions.
  • From tool - Lets you define a tool and then identifies regions that are unreachable by the defined tool.
  • From file - Lets you specify a binary STL file that represents unmachined regions.
  • From solid(s) - Lets you specify solid bodies that represent unmachined regions.
  • From setup stock - Uses the stock defined in the setup to represent unmachined regions.
