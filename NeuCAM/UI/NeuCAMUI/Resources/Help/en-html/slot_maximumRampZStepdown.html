Specifies the maximum stepdown per revolution on the ramping profile. Overrides the Angle by limiting the Z depth per pass.

Value of .750in
Ramp at the Angle until Z-.750 is reached.
Fewer passes - Heavier tool load.

Value of .250in
Ramp at the Angle until Z-.250 is reached.
More passes - Lighter tool load.

Note: This is important on long slots. This parameter allows the tool load to be constrained when doing full-width cuts during ramping.
