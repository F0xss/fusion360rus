Determines the number of axes that can be transformed when calculating the part alignment. Choose a Method that is supported by your machine tool and post processor.

  • 3 Axis: Enables translation along one, two, or three linear axes.
  • 4 Axis: Enables translation along all linear axes and rotation about one rotary axis.
  • 5 Axis: Enables translation along and rotation about all axes.

Example of 3 Axis (left), 4 Axis (center), and 5 Axis (right).

Note: The axes are based on the work coordinate system defined in the setup.
