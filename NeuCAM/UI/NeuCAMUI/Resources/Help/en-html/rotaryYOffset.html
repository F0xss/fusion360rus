Avoids using the center of a tool to cut the part.

Offsets the contact point of the tool with the part, which is the center of the tool by default, to a specified value so you can machine with the flutes of a tool instead of its center.

In the example below, the Tool Offset moves the tool from its start position in the direction of the Y axis.
