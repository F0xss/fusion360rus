Selects all holes with the same diameter as the currently selected feature. For example, if you select a single 6-mm hole and a single 12-mm hole, every 6-mm and 12-mm hole on the part is automatically selected.

A single selection can detect matching holes.

Note: This option is associative to the model. If additional holes with the same diameter are added to the model, regenerating the operation automatically includes the new holes.
