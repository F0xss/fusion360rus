Select a tool to use for trimming:

Polygon - Lets you place points to create a polygon over an area of a toolpath.
Plane - Lets you specify a plane to cover an area of a toolpath.
Boundary - Lets you select a closed contour on the model to use as a boundary over an area of a toolpath.

Planes and boundaries are associative to the model, meaning they can be regenerated if the model changes.
