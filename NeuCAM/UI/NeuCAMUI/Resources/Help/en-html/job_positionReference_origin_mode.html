Sets the attach point that is used to connect the part to the table's attach point.

Selected point - Select a point on the geometry that is visible in the canvas.

Model box point - Select a point on the model's bounding box.

Fixture box point - Select a point on the fixture's bounding box.

Stock box point - Select a point on the stock's bounding box.

Note: If Fixture box point is selected but no fixtures are defined in the Setup tab, the model's bounding box is used instead.
