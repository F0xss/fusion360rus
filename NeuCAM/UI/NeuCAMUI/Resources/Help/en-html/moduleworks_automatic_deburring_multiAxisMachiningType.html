Specifies the number of machine axes to use when deburring.

Generally, using multi-axis machining allows you to reach more areas in one operation. For 4- and 5-axis machining, you require a machine tool that is capable of multi-axis machining.

  • 3-axis machining - Useful for deburring parts that are prismatic, which has edges that are easily reachable by the tool.
  • 4-axis machining - Useful for deburring cylindrical parts.
  • 5-axis machining - Useful for deburring complex parts that require the tool to tilt to reach more areas.
