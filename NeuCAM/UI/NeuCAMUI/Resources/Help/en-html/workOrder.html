Specifies the ordering of the individual operations in this setup in the NC code.

Preserve Order – orders the operations according to the layout of the toolpaths that you have generated, and processes the operations setup by setup.

Order by Operation – switches between the multiple offsets, performing each operation on each setup before continuing to the next operation.

Order by Tool – orders the operations by loading a tool and completing all the operations for that tool in each setup before continuing to the next tool.
