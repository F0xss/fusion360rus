Specifies the angle by which the tool leans sideways in reference to the tool's direction of travel.

1 - Direction of travel
2 - Direction of sideways tilt

You can tilt the tool sideways to remove material with a larger portion of the tool and save on machining time. When a value is specified for the Sideways Tilt, the tool maintains this tilt angle as much as possible, as long as no potential collisions are detected.
