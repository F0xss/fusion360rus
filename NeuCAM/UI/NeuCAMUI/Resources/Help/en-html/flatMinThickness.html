Avoids machining flat areas that are smaller than the specified value.

The specified value is used by the strategy as the diameter of a circle that is used to evaluate the flat faces. If the circle does not fit anywhere on a flat face, the face gets filtered out.

It is useful to filter out flat areas that are long and thin, such as chamfers, when the tool you are using is unsuitable to machine chamfers.

1 mm chamfers

Filter value of 0.5 mm
Chamfers are machined

Filter value of 1.5 mm
Chamfers are avoided

Note: The filter option is available only when Machined Inclined Flats is selected in the Multi-axis tab.
