Determines what to do with areas of the toolpath that exceed the machine's tilt limits:

  • Trim toolpath - Remove the portion of the toolpath that tilts the tool beyond the tilt limits. Use this option if your tool axis angle is critical for your surface finish and then complete machining the part using another setup.
  • Machine at tilt limit - Select to keep the tool axis at the tilt limit upon reaching it. Use this option to continue machining with the toolpath in the current Setup if the tool axis angle is not critical for your surface finish.

In the example below, with Trim toolpath selected and Maximum Tilt Angle set to 90 degrees, nothing is trimmed as the tool axis always stays within 90 degrees. When the Maximum Tilt Angle is set to 65 degrees, the portion that tilts the tool axis beyond 65 degrees is trimmed.
