Illustrated with Flow


Full retraction - Completely retracts the tool to the Retract Height at the end of the pass before moving above the start of the next pass.

Minimum retraction - Moves straight up to the lowest height where the tool clears the workpiece, plus any specified safe distance.

Shortest path - Moves the tool the shortest possible distance in a straight line between paths.

WARNING: Shortest path should not be used on machines that do not support linearized rapid movements where G0 moves are straight-line (versus G0 moves that drive all axes at maximum speed, sometimes referred to as "dogleg" moves). Failure to obey this rule will result in machine motion that cannot be properly simulated by the software, and may result in tool crashes.

For CNC machines that do not support linearized rapid moves, the post processor can be modified to convert all G0 moves to high-feed G1 moves. Contact technical support for more information or instructions on how to modify post processors as described.
