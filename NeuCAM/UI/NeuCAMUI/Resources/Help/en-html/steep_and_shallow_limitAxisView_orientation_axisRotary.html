Select a plane to set the rotary axis perpendicular to. You can also select an edge to set the rotary axis parallel to. Remember that the tool axis is always perpendicular to the rotary axis.

The example below shows selecting a plane to set the rotary axis perpendicular to, and shows the resultant toolpath, which has the tool axis always parallel to the plane. The tool axis cannot tilt into or out of the plane.
